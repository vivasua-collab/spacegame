## 1. Введение

### 1.1 Проблема

При добавлении нового элемента в игру (например, теллура Te или галлия Ga) разработчик должен вручную определить:
- Какую руду образует элемент (оксид? сульфид? силикат?)
- Какое здание добывает эту руду (шахта? карьер? бурильная?)
- Какая переработка нужна (плавильня? химзавод? очистительный комплекс?)
- Какой выход из 10 ед. руды
- Сколько энергии и времени требует переработка
- Может ли элемент встречаться самородком
- Образует ли атмосферные или ледяные соединения

Каждое такое решение требует химических знаний и согласования с балансом игры.

### 1.2 Решение

Данный документ вводит **систему химических характеров** — классификацию элементов по их рудообразующему поведению. Каждый элемент относится к одному из 10 химических характеров, и для каждого характера определены **детерминированные правила** образования руд.

Это означает: при добавлении нового элемента достаточно указать его **химический характер** и **степень окисления** — и система автоматически определит все остальные параметры руды.

### 1.3 Связь с другими документами

| Документ | Роль | Что берёт из chemistry.md |

| [mendeleev.md](./mendeleev.md) | Список элементов | Химический характер и степени окисления для каждого элемента |

## 2. Химические характеры элементов

**Химический характер** — это классификация элемента по его рудообразующему поведению в игре. Он определяет:
- Какие типы соединений элемент образует в природе
- Какое здание добывает его руду
- Типичную формулу минерала-прототипа
- Может ли элемент встречаться в самородном виде

> **Важно:** химический характер ≠ игровая категория. Игровая категория (`structural`, `metal`, `noble` и т.д.) — для группировки в интерфейсе. Химический характер — для генерации руд.

### 2.1 Таблица химических характеров

| # | Характер | Описание | Типичные соединения | Здание добычи | Примеры |
|---|----------|----------|---------------------|---------------|---------|
| 1 | `reactive_metal` | Реактивный металл — образует оксиды, сульфиды, силикаты | Me₂O₃, MeO₂, MeS, MeSiO₃ | Шахта | Fe, Ti, Cu, Ni, Cr, Mn, Zn, Sn, Pb, Co, V, Al, W, Mo, Cd |
| 2 | `noble_metal` | Благородный металл — встречается в самородках и следовых концентрациях | Me (native), Me₂S, следы в сульфидах | Шахта | Au, Pt, Ag |
| 3 | `refractory_metal` | Тугоплавкий/редкий металл — оксиды из глубинных залежей | Me₂O₅, MeO₂, MeSiO₄ | Бурильная уст. | Ta, Nb, Zr, Hf, Re |
| 4 | `platinoid` | Платиноид — редчайшие благородные, глубинные сульфиды | MeS₂, Me₂S₃, MeS | Бурильная уст. | Ru, Rh, Pd, Ir, Os |
| 5 | `rare_earth` | Редкоземельный элемент — фосфаты из глубинных залежей | MePO₄ | Бурильная уст. | Y, La, Ce, Nd, Dy |
| 6 | `alkali` | Щелочной металл — галогениды, силикаты | MeCl, MeF, MeAlSi₂O₆ | Карьер / Шахта | Li, Na, K |

## 3. Степени окисления

Степень окисления определяет, в какой пропорции элемент соединяется с другими. Для расчёта молярной массы и выхода необходимо знать, сколько атомов партнёра присоединяет элемент.

### 3.1 Степени окисления по химическому характеру

| Характер | Типичная степень окисления | Формула оксида | Формула сульфида |
|----------|---------------------------|----------------|-----------------|
| `reactive_metal` | +2 или +3 | MeO, Me₂O₃ | MeS, Me₂S₃ |
| `noble_metal` | 0 (native), +1, +3 | — | Me₂S |
| `refractory_metal` | +4, +5 | MeO₂, Me₂O₅ | MeS₂ |
| `platinoid` | +2, +3, +4 | — | MeS₂, Me₂S₃, MeS |
| `rare_earth` | +3 | Me₂O₃ | — |
| `alkali` | +1 | Me₂O | — |
| `alkaline_earth` | +2 | MeO | MeS |
| `reactive_nonmetal` | −2, +3, +4, +5 | MeO₂, Me₂O₃, Me₂O₅ | — |
| `halogen` | −1 | — | — |
| `gas` | 0, −2, +4 | — | — |

### 3.2 Индивидуальные степени окисления

Для элементов с нестандартными степенями окисления указываем явно:

| Элемент | Степень окисления в руде | Примечание |
|---------|-------------------------|------------|
| Fe | +3 (в Fe₂O₃), +2 (в FeTiO₃, FeWO₄) | Две степени в разных рудах |
| Cu | +1 (в CuFeS₂) | Халькопирит — +1, не +2 |
| Ti | +4 (в FeTiO₃) | Ильменит |
| Cr | +3 (в FeCr₂O₄) | Хромит |
| V | +5 (в V₂O₅) | Концентрат |
| Al | +3 (в Al₂O₃) | Боксит |
| Sn | +4 (в SnO₂) | Касситерит |
| Pb | +2 (в PbS) | Галенит |
| U | +4 (в UO₂) | Уранинит |
| W | +6 (в FeWO₄) | Вольфрамит |
| Mo | +4 (в MoS₂) | Молибденит |
| Nb | +5 (в Nb₂O₅) | Колумбит |
| Ta | +5 (в Ta₂O₅) | Танталит |
| Zr | +4 (в ZrSiO₄) | Циркон |
| Hf | +4 (в HfSiO₄) | Аналог Zr |
| Mn | +4 (в MnO₂) | Пиролюзит |

## 4. Правила образования руд

Для каждого химического характера определён **шаблон рудообразования** — приоритетный список типов соединений, которые элемент образует в природе. Первый подходящий шаблон становится **основной рудой**.

### 4.1 `reactive_metal` — Реактивный металл

**Приоритет образования соединений:**

| Приоритет | Тип соединения | Формула | Условие | Здание |
|-----------|---------------|---------|---------|--------|
| 1 | Оксид | Me₂O₃ или MeO₂ | Если элемент образует устойчивый оксид | Шахта |
| 2 | Сульфид | MeS или Me₂S₃ | Если элемент образует устойчивый сульфид | Шахта |
| 3 | Комплексный оксид | MeFeO₃, MeFeS₂ | Многие руды содержат Fe как «разбавитель» | Шахта |
| 4 | Силикат | MeSiO₃ | Если элемент не образует чистый оксид/сульфид | Шахта |

**Выбор формулы по степени окисления:**

| Степень | Оксид | Сульфид |
|---------|-------|---------|
| +2 | MeO | MeS |
| +3 | Me₂O₃ | Me₂S₃ |
| +4 | MeO₂ | MeS₂ |

> **Исключения:** Fe (+3) → Fe₂O₃, но Fe (+2) в ильмените FeTiO₃; Cu (+1) → CuFeS₂; W (+6) → FeWO₄; V (+5) → V₂O₅

### 4.2 `noble_metal` — Благородный металл

**Особенность:** благородные металлы встречаются в **следовых концентрациях** в руде. Основная масса руды — пустая порода (SiO₂, FeS, ультрамафиты).

| Приоритет | Тип | Формула | Концентрация | Здание |
|-----------|-----|---------|-------------|--------|
| 1 | Следы в породе | Me + SiO₂ (Au), Me + FeNiS+O (Pt) | 3–5% по массе | Шахта |
| 2 | Сульфид | Me₂S (Ag) | 85–90% Me по массе | Шахта |
| 3 | Самородок | Me (native) | Шанс при добыче | — (бонус) |
| 4 | Рафинация | Me (очищенный) | ~90% через Очистительный комплекс | Очистительный |

**Разделение Ag от Au/Pt:** серебро образует устойчивый сульфид (Ag₂S) с высокой концентрацией, а золото и платина — нет.

### 4.3 `refractory_metal` — Тугоплавкий/редкий металл

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Оксид | Me₂O₅ (для +5), MeO₂ (для +4) | Бурильная уст. |
| 2 | Силикат | MeSiO₄ | Бурильная уст. |
| 3 | Сульфид | MeS₂ | Бурильная уст. (Re) |

### 4.4 `platinoid` — Платиноид

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Сульфид | MeS₂ (Ru, Ir, Os), Me₂S₃ (Rh), MeS (Pd) | Бурильная уст. |

### 4.5 `rare_earth` — Редкоземельный элемент

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Фосфат | MePO₄ | Бурильная уст. |

> **Примечание:** все редкоземельные элементы образуют один и тот же тип руды — фосфат (монацит/ксенотим). Это отражает реальность: РЗЭ всегда встречаются вместе в одних и тех же минералах.

### 4.6 `alkali` — Щелочной металл

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Галогенид | MeCl (Na, K), MeF (Li — через сподумен) | Карьер |
| 2 | Алюмосиликат | MeAlSi₂O₆ (Li — сподумен) | Шахта |

> **Исключение:** литий (Li) добывается как сложный алюмосиликат (сподумен LiAlSi₂O₆) из шахты, а не как галогенид. Это отражает реальность — хлорид лития растворим в воде и не образует осадочных руд.

### 4.7 `alkaline_earth` — Щёлочноземельный металл

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Карбонат | MeCO₃ | Карьер |
| 2 | Фторид | MeF₂ | Карьер |
| 3 | Сульфат | MeSO₄ | Карьер / Бурильная уст. |
| 4 | Силикат | Be₃Al₂Si₆O₁₈ | Бурильная уст. (Be) |

> **Разделение по элементу:** Mg, Ca → карьер (поверхностные); Ba → карьер + бурильная (двойной источник); Be → бурильная (глубинный берилл).

### 4.8 `reactive_nonmetal` — Реактивный неметалл

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Самородок | Me (native) | Карьер (S, C) |
| 2 | Оксид | MeO₂, Me₂O₃, B₂O₃ | Карьер |
| 3 | Фосфат (как примесь) | Ca₃(PO₄)₂ | Карьер |
| 4 | Селенид/теллурид | PbSe, CdTe | Шахта (как побочный продукт) |

### 4.9 `halogen` — Галоген

| Приоритет | Тип | Формула | Здание |
|-----------|-----|---------|--------|
| 1 | Галогенид с металлом | CaF₂ (F), NaCl / KCl (Cl) | Карьер |

> Галогены **никогда** не встречаются в самородном виде в природе. Они всегда связаны с металлами.

### 4.10 `gas` — Газ

| Подтип | Описание | Переработка | Примеры |
|--------|----------|-------------|---------|
| Чистый газ | Одноатомный или двухатомный | Не нужна (1:1) | H₂, He, Ne, Ar, N₂, O₂ |
| Сложный газ | Многоатомное соединение | Химзавод (разложение) | CO₂, CH₄, NH₃, H₂S, SO₂ |

> См. подробнее в §9 и [ores.md](./ores.md) §3.4–3.5.

### 4.11 `transuranic` — Трансурановый

Не образует руд в стандартном смысле. См. §10.

## 5. Матрица здание → тип руды

### 5.1 Здания добычи и химические характеры

| Здание добычи | Характеры | Тип руды | Уровень здания |
|---------------|-----------|----------|----------------|
| **Шахта** (mine) | `reactive_metal`, `noble_metal`, щелочные (Li как силикат) | `metal_ore` | 1–10 |
| **Карьер** (quarry) | `alkali` (Na, K), `alkaline_earth` (Mg, Ca, Ba), `reactive_nonmetal`, `halogen` | `nonmetal_ore` | 1–10 |
| **Бурильная уст.** (drilling_rig) | `refractory_metal`, `platinoid`, `rare_earth`, `alkaline_earth` (Be, Ba глубинный) | `deep_ore` | 1–10 |
| **Газ. экстрактор** (gas_extractor) | `gas` | `gas_compound` | 1–10 |
| **Ледодоб. станция** (ice_harvester) | `gas` (замёрзшие соединения) | `ice_compound` | 1–10 |

### 5.2 Правила выбора здания

```
Химический характер элемента
        │
        ├── gas → Газ. экстрактор / Ледодоб. станция
        │
        ├── alkali → Карьер (галогенид) или Шахта (силикат, если Li)
        │
        ├── alkaline_earth
        │       ├── Be → Бурильная уст. (силикат, глубинный)
        │       ├── Mg, Ca → Карьер (карбонат/фторид)
        │       └── Ba → Карьер + Бурильная уст. (двойной источник)
        │
        ├── reactive_nonmetal → Карьер (оксид/самородок)
        │       └── если селен/теллур → Шахта (селенид/теллурид)
        │
        ├── halogen → Карьер (галогенид)
        │
        ├── reactive_metal → Шахта
        │       └── если In (глубокий) → Бурильная уст.
        │
        ├── noble_metal → Шахта (низкая концентрация)
        │
        ├── refractory_metal → Бурильная уст.
        │
        ├── platinoid → Бурильная уст.
        │
        ├── rare_earth → Бурильная уст.
        │
        └── transuranic → Спец. здание
```

## 6. Правила назначения переработки

### 6.1 Здание переработки по типу руды

| Тип руды | Здание переработки | Мин. уровень | Исключения |
|----------|-------------------|-------------|------------|
| `metal_ore` (reactive_metal) | `processor` (Плавильня) | 1–4 | Зависит от тугоплавкости |
| `metal_ore` (noble_metal, низкий выход) | `processor` (Плавильня) | 3–5 | Альтернатива: `refinery` (ур. 3–5) |
| `deep_ore` (refractory_metal) | `processor` (Плавильня) | 5–8 | Высокий уровень из-за тугоплавкости |
| `deep_ore` (platinoid) | `processor` (Плавильня) | 6–7 | |
| `deep_ore` (rare_earth) | `processor` (Плавильня) | 5–6 | |
| `nonmetal_ore` | `processor` (Плавильня/Химзавод) | 1–3 | |
| `gas_compound` (чистый газ) | `null` (не нужна) | — | H₂, He, Ne, Ar, N₂, O₂ |
| `gas_compound` (сложный газ) | `processor` (Химзавод) | 1–2 | CO₂, CH₄, NH₃, H₂S, SO₂ |
| `ice_compound` (чистый) | `null` (не нужна) | — | N₂-ice |
| `ice_compound` (сложный) | `processor` (Химзавод) | 1–2 | H₂O, CO₂, CH₄, NH₃ |

### 6.2 Правило уровня переработки

Уровень переработки зависит от **сложности извлечения** элемента из соединения:

| Сложность | Мин. уровень | Критерий |
|-----------|-------------|----------|
| Лёгкая | 1 | Распространённые элементы (Fe, Si, Al, Cu) |
| Средняя | 2–3 | Легирующие элементы (Cr, Mn, Ni, Zn, Sn, Pb) |
| Сложная | 3–4 | Редкие/тугоплавкие (Co, V, W, Mo, Ag, Cd, Se) |
| Очень сложная | 5–6 | Благородные (Au, Pt, U) / РЗЭ / Платиноиды |
| Экстремальная | 7–8 | Рений (Re), иридий (Ir), осмий (Os) |

### 6.3 Очистительный комплекс как альтернатива

Для благородных и стратегических металлов (Au, Pt, U) Очистительный комплекс даёт **значительно более высокий выход** при более высоких требованиях:

| Элемент | Плавильня (выход) | Очистительный комплекс (выход) | Мин. уровень Очист. |
|---------|-------------------|-------------------------------|---------------------|
| Au | 0.4 из 10 | 9 из 10 | 3 |
| Pt | 0.3 из 10 | 9 из 10 | 5 |
| U | 8.8 из 10 | 9 из 10 | 5 |

## 7. Расчёт энергозатрат и времени

### 7.1 Базовые формулы

Энергозатраты и время переработки зависят от:
1. **Химического характера** — более «реактивные» элементы требуют меньше энергии
2. **Типа руды** — глубинные руды дороже в переработке
3. **Редкости элемента** — редкие элементы требуют больше энергии

### 7.2 Таблица базовых энергозатрат

| Характер | Тип руды | Базовая энергия | Базовое время (тики) |
|----------|----------|----------------|---------------------|

| `noble_metal` | metal_ore | 5–8 | 250–300 |
| `refractory_metal` | deep_ore | 6–12 | 300–400 |
| `platinoid` | deep_ore | 8–10 | 300–350 |
| `rare_earth` | deep_ore | 6–9 | 300–350 |
| `alkali` | nonmetal_ore | 2–4 | 150–200 |
| `alkaline_earth` | nonmetal_ore | 2–5 | 180–250 |
| `reactive_nonmetal` | nonmetal_ore | 1–4 | 100–250 |
| `halogen` | nonmetal_ore | 2–4 | 150–200 |
| `gas` (сложный) | gas_compound | 2–5 | 150–200 |
| `ice` (сложный) | ice_compound | 3–5 | 180–200 |

### 7.3 Поправочный коэффициент по редкости

| Редкость | Поправка энергии | Примеры |
|----------|-----------------|---------|

## 8. Самородные элементы

### 8.1 Правила самородного нахождения

Элемент может встречаться в самородном (чистом) виде, если выполняется **хотя бы одно** из условий:

| Условие | Характер | Примеры |
|---------|----------|---------|
| Элемент — благородный металл с низкой реактивностью | `noble_metal` | Au, Pt, Ag |
| Элемент — неметалл с низкой реактивностью при стандартных условиях | `reactive_nonmetal` | S, C |
| Элемент имеет высокое сродство к самому себе (металлическая связь) | `reactive_metal` | Cu (редко) |

### 8.2 Расчёт шанса самородка

```
nativeChance = baseNativeChance + buildingLevel × 0.02
```

| Характер | baseNativeChance | Обоснование |
|----------|-----------------|-------------|
| `noble_metal` (Au, Pt) | 0.03–0.10 | Благородные металлы часто самородны |
| `reactive_nonmetal` (S) | 0.30 | Сера — самый частый самородок |
| `reactive_nonmetal` (C) | 0.20 | Графит/алмаз — самородный углерод |
| `noble_metal` (Ag) | 0.05 | Серебро — реже самородное |
| `reactive_metal` (Cu) | 0.05 | Медь — редко самородная |
| Все остальные | 0.00 | Не встречаются самородно |

### 8.3 Элементы, НЕ встречающиеся самородно

- **Щелочные и щёлочноземельные** (Li, Na, K, Mg, Ca, Ba, Be) — слишком реактивны
- **Галогены** (F, Cl) — всегда в виде галогенидов
- **Газы** (H, He, N, O) — добываются как соединения из атмосферы/льдов
- **Редкоземельные** (Y, La, Ce, Nd, Dy) — всегда в виде фосфатов
- **Тугоплавкие/платиноиды из глубин** (Zr, Hf, Ta, Nb, Ru, Rh, Pd, Ir, Os, Re) — всегда в виде соединений

## 9. Атмосферные и ледяные соединения

### 9.1 Критерий принадлежности к атмосферным

Элемент или его соединение является атмосферным, если:
- Элемент — газ при стандартных условиях (`gas` характер)
- Соединение имеет точку кипения ниже 0°C
- Соединение входит в состав реальных атмосфер планет

### 9.2 Чистые газы (переработка не нужна)

| Газ | Формула | Преобразование | Выход из 10 ед. |
|-----|---------|----------------|-----------------|
| Водород | H₂ | H₂ → H (1:1) | H ≈ 10 |
| Гелий | He | He → He | He ≈ 10 |
| Неон | Ne | Ne → Ne | Ne ≈ 10 |
| Аргон | Ar | Ar → Ar | Ar ≈ 10 |
| Азот | N₂ | N₂ → N (1:1) | N ≈ 10 |
| Кислород | O₂ | O₂ → O (1:1) | O ≈ 10 |

### 9.3 Сложные газы (требуют разложения)

| Газ | Формула | Компоненты | Выход из 10 ед. (химически корректный) |
|-----|---------|------------|---------------------------------------|
| Углекислый газ | CO₂ | C + O | C ≈ 2.7, O ≈ 7.3 |
| Метан | CH₄ | C + H | C ≈ 7.5, H ≈ 2.5 |
| Аммиак | NH₃ | N + H | N ≈ 8.2, H ≈ 1.8 |
| Сероводород | H₂S | H + S | H ≈ 0.6, S ≈ 9.4 |
| Диоксид серы | SO₂ | S + O | S ≈ 5.0, O ≈ 5.0 |

> **⚠️ Балансное замечание:** текущий код использует «балансные» значения, где сумма выходов > 10 (напр. NH₃: N≈8, H≈12, сумма 20). Химически корректные значения дают сумму = 10. Решение о переходе на химически корректные значения — отдельная задача баланса.

### 9.4 Ледяные соединения

Ледяные соединения — это замёрзшие атмосферные газы. Состав идентичен газовой фазе, но добыча зависит от температуры планеты.

| Лёд | Формула | Макс. T (°C) | Компоненты | Выход |
|-----|---------|-------------|------------|-------|
| Водяной лёд | H₂O | +50 | H + O | H ≈ 1.1, O ≈ 8.9 |
| Сухой лёд | CO₂ | −50 | C + O | C ≈ 2.7, O ≈ 7.3 |
| Аммиачный лёд | NH₃ | −100 | N + H | N ≈ 8.2, H ≈ 1.8 |
| Азотный лёд | N₂ | −150 | N | N ≈ 10 |
| Метановый лёд | CH₄ | −150 | C + H | C ≈ 7.5, H ≈ 2.5 |

### 9.5 Доступность по типу атмосферы

См. [ores.md](./ores.md) §3.4.12 — полная матрица «тип атмосферы → доступные газы».

## 10. Трансурановые элементы — особые правила

Трансурановые элементы **не образуют руд** в стандартном смысле. Они добываются принципиально иными методами, связанными с экстремальными космическими объектами.

### 10.1 Уровни добычи

| Уровень | Объект | Элементы | Механика |
|---------|--------|----------|----------|
| I | Нейтронная звезда | Np, Pu, Am, Cm, Cf | Синтез из нейтронного потока |
| II | Звезда (O/B/A класс) | Fl, Og | Звёздная плавильня, магнитная сепарация |
| III | Пульсар / Чёрная дыра | Xn, Qn, Vd | Магнитная стабилизация / извлечение из эргосферы |

### 10.2 Отличия от стандартных элементов

| Свойство | Стандартные элементы | Трансурановые |
|----------|---------------------|---------------|
| Руда | Минерал с молярной массой | Нет руды — синтез/извлечение |
| Здание добычи | Шахта/Карьер/Бурильная | Нейтронный сборщик / Звёздная плавильня / Сингулярный сборщик |
| Переработка | Плавильня/Химзавод | Не нужна (сразу чистый элемент) |
| Устойчивость | Стабильны | Требуют стабилизации (Xn, Qn, Vd) |
| Молярная масса | Точная | Приблизительная (теоретическая) |

> Детальная механика трансурановых элементов — в [mendeleev.md](./mendeleev.md) §2.11.

## 11. Алгоритм автогенерации руды для нового элемента

При добавлении нового элемента система автоматически определяет все параметры руды по следующему алгоритму:

### Шаг 1: Определить химический характер

На основе позиции элемента в таблице Менделеева и его свойств:

```
if (элемент — газ при станд. условиях) → gas
elif (Z > 92) → transuranic
elif (элемент — благородный металл, низкая реактивность) → noble_metal
elif (элемент — платиноид, глубинный) → platinoid
elif (элемент — РЗЭ, лантаноид) → rare_earth
elif (элемент — тугоплавкий, глубинный) → refractory_metal
elif (элемент — щелочной металл) → alkali
elif (элемент — щёлочноземельный) → alkaline_earth
elif (элемент — галоген) → halogen
elif (элемент — неметалл, не газ) → reactive_nonmetal
else → reactive_metal
```

### Шаг 2: Определить тип руды и здание добычи

По таблице из §5.2:

```
chemicalCharacter → buildingId + oreType
```

### Шаг 3: Выбрать формулу минерала-прототипа

По шаблону из §4 для данного химического характера:

```
reactive_metal + степень +3 → Me₂O₃ (оксид)
reactive_metal + степень +2 → MeS (сульфид)
rare_earth → MePO₄ (фосфат)
refractory_metal + степень +5 → Me₂O₅ (оксид)
...
```

### Шаг 4: Рассчитать молярную массу

```
M(соед) = Σ(nᵢ × Mᵢ)
```

Где Mᵢ берутся из [mendeleev.md](./mendeleev.md) §2.12.

### Шаг 5: Рассчитать выход

```
wᵢ = (nᵢ × Mᵢ) / M(соед)
outputᵢ = 10 × wᵢ  (при efficiency = 1.0)
```

### Шаг 6: Назначить переработку

По таблице из §6:

```
oreType + chemicalCharacter → processingBuildingId + minProcessingLevel
```

### Шаг 7: Рассчитать энергозатраты и время

По таблице из §7 с поправкой на редкость:

```
energyCost = baseEnergy × rarityModifier
processingTime = baseTime × rarityModifier
```

### Шаг 8: Определить самородный шанс

По таблице из §8:

```
if (характер ∈ {noble_metal, reactive_nonmetal (S, C), reactive_metal (Cu)})
    → nativeChance = baseNativeChance
else → nativeChance = 0
```

### Шаг 9: Проверить атмосферные/ледяные соединения

```
if (характер == gas)
    → добавить в ATMOSPHERIC_COMPOUNDS и/или ICE_COMPOUNDS
    → определить доступность по типам атмосферы
```

## 12. Примеры автогенерации

### 12.1 Пример: добавление галлия (Ga)

**Входные данные:**
- Символ: Ga, атомная масса: 69.7
- Игровая категория: `metal`
- Степень окисления: +3

**Шаг 1:** Химический характер → `reactive_metal` (металл, не благородный, не тугоплавкий)

**Шаг 2:** Здание добычи → `mine` (шахта), тип → `metal_ore`

**Шаг 3:** Формула → Ga₂O₃ (оксид галлия)

**Шаг 4:** Молярная масса → M(Ga₂O₃) = 2 × 69.7 + 3 × 16.0 = 139.4 + 48.0 = **187.4 г/моль**

**Шаг 5:** Выход:
- w(Ga) = 139.4 / 187.4 = **74.4%**
- w(O) = 48.0 / 187.4 = **25.6%**
- Выход из 10 ед.: **Ga ≈ 7.4, O ≈ 2.6**

**Шаг 6:** Переработка → `processor`, мин. уровень 2

**Шаг 7:** Энергозатраты → базовая 3 (reactive_metal) × 1.2 (common) ≈ **4**, время ≈ **200**

**Шаг 8:** nativeChance = 0 (не благородный)

**Результат:**
```typescript
{
  id: 'Ga-ore', name: 'Галлиевая руда', type: 'metal_ore',
  sourceBuildingId: 'mine',
  containedElements: [
    { elementId: 'Ga', yield: 7.4 },
    { elementId: 'O', yield: 2.6 },
  ],
  minSourceLevel: 1, processingBuildingId: 'processor', minProcessingLevel: 2,
  processingEnergyCost: 4, processingTime: 200,
  prototype: 'Оксид галлия (Ga₂O₃)',
  molarFormula: 'Ga₂O₃', molarMass: 187.4,
}
```

### 12.2 Пример: добавление эрбия (Er)

**Входные данные:**
- Символ: Er, атомная масса: 167.3
- Игровая категория: `lanthanide`
- Степень окисления: +3

**Шаг 1:** Химический характер → `rare_earth`

**Шаг 2:** Здание добычи → `drilling_rig`, тип → `deep_ore`

**Шаг 3:** Формула → ErPO₄ (фосфат эрбия)

**Шаг 4:** Молярная масса → M(ErPO₄) = 167.3 + 31.0 + 4 × 16.0 = 167.3 + 31.0 + 64.0 = **262.3 г/моль**

**Шаг 5:** Выход:
- w(Er) = 167.3 / 262.3 = **63.8%**
- w(P) = 31.0 / 262.3 = **11.8%**
- w(O) = 64.0 / 262.3 = **24.4%**
- Выход из 10 ед.: **Er ≈ 6.4, P ≈ 1.2, O ≈ 2.4**

**Шаг 6:** Переработка → `processor`, мин. уровень 5

**Шаг 7:** Энергозатраты → базовая 7 (rare_earth) × 1.5 (rare) ≈ **8**, время ≈ **300**

**Шаг 8:** nativeChance = 0

**Результат:**
```typescript
{
  id: 'Er-ore', name: 'Эрбиевая руда', type: 'deep_ore',
  sourceBuildingId: 'drilling_rig',
  containedElements: [
    { elementId: 'Er', yield: 6.4 },
    { elementId: 'P', yield: 1.2 },
    { elementId: 'O', yield: 2.4 },
  ],
  minSourceLevel: 5, processingBuildingId: 'processor', minProcessingLevel: 5,
  processingEnergyCost: 8, processingTime: 300,
  prototype: 'Фосфат эрбия (ErPO₄)',
  molarFormula: 'ErPO₄', molarMass: 262.3,
}
```

### 12.3 Пример: добавление брома (Br)

**Входные данные:**
- Символ: Br, атомная масса: 79.9
- Игровая категория: `halogen`
- Степень окисления: −1

**Шаг 1:** Химический характер → `halogen`

**Шаг 2:** Здание добычи → `quarry`, тип → `nonmetal_ore`

**Шаг 3:** Формула → NaBr или KBr (бромид) — аналогично NaCl

**Шаг 4:** Молярная масса → M(KBr) = 39.1 + 79.9 = **119.0 г/моль**

**Шаг 5:** Выход:
- w(K) = 39.1 / 119.0 = **32.9%**
- w(Br) = 79.9 / 119.0 = **67.1%**
- Выход из 10 ед.: **K ≈ 3.3, Br ≈ 6.7**

**Шаг 6:** Переработка → `processor`, мин. уровень 2

**Шаг 7:** Энергозатраты → базовая 3 × 1.5 = **4**, время ≈ **200**

**Шаг 8:** nativeChance = 0 (галогены не самородны)

**Результат:**

    { elementId: 'K', yield: 3.3 },
    { elementId: 'Br', yield: 6.7 },

  processingEnergyCost: 4, processingTime: 200,
