## 1. Введение

### 1.1 Назначение документа

Данный документ — **справочник химических элементов** для игры SpaceGame. Для каждого элемента указаны:

- **Атомная масса** — используется для расчёта молярной массы минералов
- **Категория** — группировка элементов в интерфейсе игры
- **Описание** — реальное применение элемента и его роль в космической индустрии
- **Химические взаимодействия** — ключевые реакции, определяющие в каких минералах/рудах элемент встречается

### 1.2 Связь с рудами и расчётами

Расчёт состава руд на основе молярных масс элементов — см. [ores.md](./ores.md).  
Принцип модульности и автоматическое встраивание новых элементов — см. [modularity.md](./modularity.md).

### 2.11 Трансурановые и экзотические элементы (10)

- Пульсары: экстремальные магнитные поля (10⁸–10¹¹ Тл) удерживают электроны на орбитах, предотвращая распад

> - **Воидий (Vd):** Добывается из аккреционного диска чёрной дыры «Сингулярным сборщиком». Самый опасный элемент — при потере стабилизации создаёт микросингулярность. Требует технологии containment level 5 (максимальный).
