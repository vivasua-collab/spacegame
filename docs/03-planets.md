## 1. Типы планет

Каждая планета в игре принадлежит к одному из семи типов. Тип определяет диапазоны генерируемых параметров, типы местности на гексагональной сетке, распределение ресурсов и базовую габитабельность.

### 1.1 Сводная таблица типов планет

| Тип | Радиус (км) | Гравитация (g) | Температура (°C) | Вероятность атмосферы | Габитабельность | Класс размера сетки |
|-----|-------------|----------------|-------------------|-----------------------|-----------------|---------------------|
| Скалистая | 2 000 – 7 000 | 0.3 – 1.4 | −50 – +150 | 40% | 40 – 90% | Малая / Средняя / Большая |
| Вулканическая | 2 500 – 8 000 | 0.4 – 1.8 | +200 – +800 | 60% | 0 – 15% | Малая / Средняя / Большая |
| Ледяная | 1 500 – 6 000 | 0.1 – 1.0 | −230 – −30 | 20% | 5 – 25% | Крошечная / Малая / Средняя |
| Океаническая | 4 000 – 8 000 | 0.7 – 1.4 | −10 – +60 | 85% | 60 – 95% | Средняя / Большая |
| Пустынная | 2 000 – 6 500 | 0.2 – 1.2 | +30 – +250 | 15% | 5 – 30% | Малая / Средняя |
| Газовый гигант | 25 000 – 80 000 | 1.5 – 3.0 | −180 – +1 000 | 100% | 0% | 0 гексов поверхности (см. слои застройки §3.2) |
| Карликовая | 500 – 2 000 | 0.05 – 0.3 | −230 – +50 | 10% | 0 – 20% | Крошечная |

### 1.2 Подробное описание каждого типа

#### 1.2.1 Скалистая (Rocky)

Стандартный мир для колонизации. Наиболее сбалансированный тип планеты с умеренными параметрами и разнообразными ресурсами. Оптимальный кандидат для первой колонии.

| Параметр | Значение |
|----------|----------|
| Радиус | 2 000 – 7 000 км |
| Гравитация | 0.3 – 1.4 g |
| Температура поверхности | −50 – +150 °C |
| Вероятность атмосферы | 40% |
| Типы атмосферы при наличии | Тонкая (40%), Стандартная (35%), Плотная (15%), Инертная (10%) |
| Габитабельность | 40 – 90% (базовая 65%) |
| Магнитное поле | Вероятность 50% |
| Наличие жизни | Нет (50%), Микробы (30%), Растения (15%), Простая (5%) |
| Сложные органические соединения | Вероятность 15% |
| Преобладающие типы местности | Равнина (40%), Горы (30%), Пустыня (20%), Джунгли (10%) |
| Ключевые ресурсы | Si, Al, O, C, Ca, Fe — стандартный набор для строительства |
| Редкие ресурсы | Ti, Cr, Cu, Zn — с пониженной доступностью |

**Игровая роль**: Основной тип планет для колонизации. Скалистые миры — «хлеб» империи: на них проще всего развернуть промышленность, они дают стабильный приток базовых ресурсов. Служат отправной точкой для дальнейшей экспансии.

---

#### 1.2.2 Вулканическая (Volcanic)

Мир с интенсивной тектонической активностью. Поверхность покрыта лавовыми полями и вулканами. Экстремальные условия, но богатейшие залежи тяжёлых металлов и геотермальная энергия.

| Параметр | Значение |
|----------|----------|
| Радиус | 2 500 – 8 000 км |
| Гравитация | 0.4 – 1.8 g |
| Температура поверхности | +200 – +800 °C |
| Вероятность атмосферы | 60% |
| Типы атмосферы при наличии | Плотная (30%), Токсичная (45%), Инертная (15%), CO2 (10%) |
| Габитабельность | 0 – 15% (базовая 5%) |
| Магнитное поле | Вероятность 70% |
| Наличие жизни | Нет (95%), Микробы-экстремофилы (5%) |
| Сложные органические соединения | Вероятность 0% |
| Преобладающие типы местности | Вулкан (45%), Горы (30%), Равнина (25%) |
| Ключевые ресурсы | Fe, Ti, Cr, Ni, W, S, V, Mo — тяжёлые металлы и сера |
| Бонус геотермальной энергии | ×2.0 к выходу геотермальных электростанций |

**Игровая роль**: Промышленная «кузница» — здесь добываются редкие и тугоплавкие металлы, необходимые для высокотехнологичных компонентов (сверхпроводники, бронеплиты, оружейные системы). Геотермальная энергия обеспечивает дешёвое электричество. Колонизация требует серьёзных инвестиций в герметизацию и теплоизоляцию.

---

#### 1.2.3 Ледяная (Ice)

Мир, покрытый ледниками и замёрзшими океанами. Беден тяжёлыми металлами, но богат летучими элементами и водой — критическими ресурсами для поддержания колоний и производства ракетного топлива.

| Параметр | Значение |
|----------|----------|
| Радиус | 1 500 – 6 000 км |
| Гравитация | 0.1 – 1.0 g |
| Температура поверхности | −230 – −30 °C |
| Вероятность атмосферы | 20% |
| Типы атмосферы при наличии | Тонкая (50%), Инертная (30%), CO2 (20%) |
| Габитабельность | 5 – 25% (базовая 10%) |
| Магнитное поле | Вероятность 25% |
| Наличие жизни | Нет (85%), Микробы (15%) |
| Сложные органические соединения | Вероятность 5% |
| Преобладающие типы местности | Лёд (50%), Горы (25%), Равнина (25%) |
| Ключевые ресурсы | H, O (вода), He, N, Ne — лёткие элементы и вода |
| Бонус добычи воды | ×1.5 к скорости добычи H₂O |

**Игровая роль**: «Водохранилище» империи. Ледяные миры снабжают колонии водой и летучими элементами. Вода нужна для жизнеобеспечения колоний, водород — для ракетного топлива и термоядерного синтеза. Часто малые и карликовые — ограниченная застройка компенсируется ценностью ресурсов.

---

#### 1.2.4 Океаническая (Oceanic)

Мир, поверхность которого почти полностью покрыта океаном. Наиболее подходящий тип для жизни земного типа. Огромные запасы воды, богатая морская экосистема.

| Параметр | Значение |
|----------|----------|
| Радиус | 4 000 – 8 000 км |
| Гравитация | 0.7 – 1.4 g |
| Температура поверхности | −10 – +60 °C |
| Вероятность атмосферы | 85% |
| Типы атмосферы при наличии | Стандартная (55%), Плотная (25%), Тонкая (15%), Инертная (5%) |
| Габитабельность | 60 – 95% (базовая 80%) |
| Магнитное поле | Вероятность 75% |
| Наличие жизни | Нет (5%), Микробы (10%), Растения (30%), Простая (35%), Сложная (20%) |
| Сложные органические соединения | Вероятность 40% |
| Преобладающие типы местности | Океан (65%), Джунгли (15%), Равнина (15%), Горы (5%) |
| Ключевые ресурсы | H, O, Na, Cl, Mg, Ca — морские ресурсы и соль |
| Бонус био-исследований | ×1.5 к скорости биологических исследований при наличии жизни |

**Игровая роль**: «Жемчужина» империи — наивысшая габитабельность, бонус к био-исследованиям, возможность добычи из океана. Ограниченная площадь суши (35% гексов — суша, 65% — океан) означает меньше места для наземных построек, но океанские гексы позволяют строить платформы и добывать морские ресурсы. Идеальный кандидат для столицы и научных центров.

---

#### 1.2.5 Пустынная (Desert)

Засушливый мир с редкими осадками. Эрозия обнажает минеральные залежи, делая редкие элементы доступнее, чем на других типах планет. Однако критический дефицит воды ограничивает колониальный потенциал.

| Параметр | Значение |
|----------|----------|
| Радиус | 2 000 – 6 500 км |
| Гравитация | 0.2 – 1.2 g |
| Температура поверхности | +30 – +250 °C |
| Вероятность атмосферы | 15% |
| Типы атмосферы при наличии | Тонкая (40%), CO2 (35%), Токсичная (15%), Инертная (10%) |
| Габитабельность | 5 – 30% (базовая 15%) |
| Магнитное поле | Вероятность 30% |
| Наличие жизни | Нет (75%), Микробы (20%), Растения (5%) |
| Сложные органические соединения | Вероятность 3% |
| Преобладающие типы местности | Пустыня (55%), Горы (25%), Равнина (20%) |
| Ключевые ресурсы | Si, Fe, Al, Ti, Rare Earth (La, Ce, Nd) — эрозия обнажает залежи |
| Бонус доступности редких элементов | ×1.3 к доступности редких элементов |

**Игровая роль**: «Редкоземельная шахта» — пустынные миры часто содержат редкоземельные элементы и другие ценные минералы, которые сложно найти на других типах планет. Дефицит воды требует импорта или строительства опреснительных установок. Подходят для специализированных добывающих колоний.

---

#### 1.2.6 Газовый гигант (Gas Giant)

Массивное небесное тело с плотной атмосферой, состоящей преимущественно из водорода и гелия. Поверхности в привычном понимании нет — поверхностная застройка невозможна (0 гексов). Однако газовый гигант имеет два доступных слоя застройки вместо поверхности: **верхняя атмосфера** (6–12 слотов для атмосферных платформ и газовых экстракторов, см. §3.2.2) и **орбита** (6–12 слотов для орбитальных станций, см. §3.2.3). Атмосфера — источник лётких элементов, а система спутников часто содержит ценные ресурсы.

| Параметр | Значение |
|----------|----------|
| Радиус | 25 000 – 80 000 км |
| Гравитация | 1.5 – 3.0 g (на условной «поверхности» облаков) |
| Температура «поверхности» | −180 – +1 000 °C (варьируется по глубине) |
| Вероятность атмосферы | 100% (всегда есть) |
| Типы атмосферы | Плотная (60%), Стандартная (10%), Метановая (15%), CO2 (10%), Инертная (5%) |
| Габитабельность | 0% (всегда) |
| Магнитное поле | Вероятность 90% (мощное) |
| Наличие жизни | Нет (100%) |
| Сложные органические соединения | Вероятность 0% |
| Застройка поверхности | Невозможна (0 гексов) — застройка осуществляется в слоях верхней атмосферы и орбиты (см. §3.2) |
| Слой верхней атмосферы | 6–12 слотов (атмосферные платформы, газовые экстракторы) |
| Орбитальный слой | 6–12 слотов (орбитальные станции, верфи) |
| Ключевые атмосферные ресурсы | H, He, C, N, CH₄, NH₃ — добыча из атмосферы спец. оборудованием |
| Количество спутников | 2 – 12 (генерируются отдельно, см. раздел 1.3) |

**Игровая роль**: «Топливная база» — газовые гиганты — главный источник водорода для термоядерного синтеза и гелия-3 для передовых реакторов. Добыча из атмосферы осуществляется через слой верхней атмосферы (атмосферные платформы) и орбитальные зонды (см. §3.2). Спутники газовых гигантов — часто ценные скалистые или ледяные миры-спутники с собственными ресурсами.

---

#### 1.2.7 Карликовая (Dwarf)

Маленькое небесное тело — от крупного астероида до небольшой планеты. Ограниченная площадь застройки, слабая гравитация, но дешёвое строительство и отсутствие атмосферного сопротивления.

| Параметр | Значение |
|----------|----------|
| Радиус | 500 – 2 000 км |
| Гравитация | 0.05 – 0.3 g |
| Температура поверхности | −230 – +50 °C |
| Вероятность атмосферы | 10% |
| Типы атмосферы при наличии | Тонкая (70%), Инертная (30%) |
| Габитабельность | 0 – 20% (базовая 5%) |
| Магнитное поле | Вероятность 5% |
| Наличие жизни | Нет (98%), Микробы (2%) |
| Сложные органические соединения | Вероятность 1% |
| Преобладающие типы местности | Равнина (50%), Горы (30%), Лёд (20%) |
| Ключевые ресурсы | Fe, Ni, Si, C — базовые элементы, иногда Pt, Ir (если астероидного происхождения) |
| Бонус строительства | −30% к стоимости зданий (низкая гравитация) |

**Игровая роль**: «Автоматизированная шахта» — карликовые миры не подходят для больших колоний, но идеальны для автоматизированных добывающих станций. Низкая гравитация удешевляет строительство и запуск грузов. Часто первый тип планеты, который колонизируется вне домашней системы.

### 1.3 Спутники планет

Спутники генерируются для планет-гигантов и больших планет (радиус > 5 000 км). Каждый спутник — самостоятельная планета карликового или малого размера с собственными параметрами.

| Параметр генерации | Значение |
|---------------------|----------|
| Количество спутников у газового гиганта | 2 – 12 (случайно, вес в сторону 4–6) |
| Количество спутников у большой планеты | 0 – 3 |
| Тип спутника | Скалистая (30%), Ледяная (40%), Карликовая (30%) |
| Радиус спутника | 200 – 2 500 км |
| Ресурсы спутника | Генерируются отдельно по типу |

**Примечание для программиста**: спутники хранятся как дочерние объекты планеты. Каждый спутник имеет полный набор параметров Planet (кроме собственных спутников), но размер сетки — всегда Крошечная (19 гексов) или Малая (37 гексов). Орбитальные станции строятся вокруг родительской планеты и доступны для всех её спутников.

## 2. Параметры планеты

### 2.1 Радиус

**Диапазон**: 500 – 80 000 км (зависит от типа планеты)

Радиус определяет класс размера планеты и, соответственно, размер гексагональной сетки застройки. Радиус указывается в Earth-радиусах (R⊕): 1 R⊕ = 6 371 км.

| Класс размера | Радиус планеты (R⊕) | Количество гексов | Пример |
|---------------|----------------------|-------------------|--------|
| Крошечная | R < 0.3 | 19 | Меркурий, Плутон |
| Малая | 0.3 ≤ R < 0.7 | 37 | Марс |
| Средняя | 0.7 ≤ R < 1.3 | 61 | Земля, Венера |
| Большая | 1.3 ≤ R < 2.0 | 91 | Суперземля |
| Огромная | R ≥ 2.0 | 127 | Мегаземля |
| Газовый гигант | — | 0 (поверхность невозможна; см. слои застройки §3.2) | Юпитер, Сатурн |

> **Единственный источник истины для размеров планет — данный файл (03-planets.md). Файл 04-buildings.md содержит ссылки сюда.**

**Формула для программиста**:
```typescript
function getGridSize(planetType: PlanetType, radiusInEarthRadii: number): number {
  if (planetType === 'gas_giant') return 0;  // Поверхность невозможна
  
  const R = radiusInEarthRadii;
  
  if (R < 0.3) return 19;    // Крошечная
  if (R < 0.7) return 37;    // Малая
  if (R < 1.3) return 61;    // Средняя
  if (R < 2.0) return 91;    // Большая
  return 127;                 // Огромная
}
```

---

### 2.2 Гравитация

**Диапазон**: 0.05 – 3.0 g (зависит от типа планеты)

Гравитация — один из ключевых параметров, влияющий на множество аспектов колониальной жизни.

| Диапазон | Класс | Влияние |
|----------|-------|---------|
| 0.05 – 0.2 g | Микрогравитация | −40% стоимость строительства, −30% скорость добычи, +50% скорость строительства, нельзя строить тяжёлые здания |
| 0.2 – 0.5 g | Низкая | −20% стоимость строительства, −15% скорость добычи, +25% скорость строительства |
| 0.5 – 1.2 g | Оптимальная | Без штрафов и бонусов (базовые значения) |
| 1.2 – 2.0 g | Повышенная | +20% стоимость строительства, +15% скорость добычи, −15% скорость строительства |
| 2.0 – 3.0 g | Высокая | +40% стоимость строительства, +30% скорость добыча, −30% скорость строительства, штраф к здоровью колонистов |

**Формулы для программиста**:
```
function getGravityModifier(gravity: number): GravityModifiers {
  const g = gravity;
  return {
    buildCost:    g < 0.2 ? 0.6 : g < 0.5 ? 0.8 : g < 1.2 ? 1.0 : g < 2.0 ? 1.2 : 1.4,
    miningSpeed:  g < 0.2 ? 0.7 : g < 0.5 ? 0.85 : g < 1.2 ? 1.0 : g < 2.0 ? 1.15 : 1.3,
    buildSpeed:   g < 0.2 ? 1.5 : g < 0.5 ? 1.25 : g < 1.2 ? 1.0 : g < 2.0 ? 0.85 : 0.7,
    colonistHealth: g < 0.05 ? 0.5 : g < 0.2 ? 0.8 : g < 2.0 ? 1.0 : 0.7,
  };
}
```

**Связь гравитации и радиуса**: при генерации гравитация вычисляется из радиуса и плотности планеты:
```
gravity = (radius / earthRadius) × (density / earthDensity)
```
где `earthRadius = 6371 км`, `earthDensity = 5.51 г/см³`. Плотность зависит от типа планеты (см. таблицу ниже).

> **Вывод формулы**: g = GM/r² = G(4/3)πρr³/r² = (4/3)πGρr. Нормализуя к Земле: g/g_E = (ρ/ρ_E)(r/r_E). Степень при r/r_E — **первая**, а не вторая.

> **Проверка (Марс)**: r = 3390 км, ρ = 3.93 г/см³ → g = (3390/6371) × (3.93/5.51) = 0.532 × 0.713 = 0.38 g ✓

> **Примечание (Р2-17)**: Данные §1.2.6 (гравитация газового гиганта 1.5–3.0 g) согласуются с правильной формулой. При ошибочной формуле с квадратом газовый гигант с r ≈ 11 R⊕ и ρ ≈ 1.5 давал бы g ≈ 11² × 0.27 ≈ 33 g — абсурд. С линейной формулой: g ≈ 11 × 0.27 ≈ 3.0 g — корректно.

| Тип планеты | Средняя плотность (г/см³) | Диапазон плотности |
|-------------|--------------------------|--------------------|
| Скалистая | 5.0 | 3.5 – 6.5 |

### 2.3 Температура поверхности

**Диапазон**: −230 – +1 000 °C (зависит от типа планеты)

Температура влияет на типы доступных зданий, эффективность солнечных электростанций и здоровье колонистов.

| Диапазон | Класс | Влияние |
|----------|-------|---------|
| −230 – −100 °C | Экстремальный холод | −40% эффективность зданий, −50% эффективность солнечных панелей, требует подогрева |
| −100 – −30 °C | Холод | −20% эффективность зданий, −30% эффективность солнечных панелей |
| −30 – +10 °C | Прохладно | −10% эффективность зданий, базовая эффективность солнечных панелей |
| +10 – +35 °C | Умеренно | Без штрафов, +10% эффективность солнечных панелей, оптимально для колонистов |
| +35 – +60 °C | Тепло | −5% эффективность зданий, +15% эффективность солнечных панелей |
| +60 – +150 °C | Жарко | −15% эффективность зданий, +20% эффективность солнечных панелей, штраф к здоровью |
| +150 – +800 °C | Экстремальная жара | −40% эффективность зданий, +30% эффективность солнечных панелей, требуется теплоизоляция |
| 800+ °C | Расплав | Здания невозможно строить (только вулканические планеты), только автоматические зонды |

**Формула для программиста**:
```
function getTemperatureModifier(tempC: number): TemperatureModifiers {
  return {
    buildingEfficiency:
      tempC < -100 ? 0.6 :
      tempC < -30  ? 0.8 :
      tempC < 10   ? 0.9 :
      tempC < 35   ? 1.0 :
      tempC < 60   ? 0.95 :
      tempC < 150  ? 0.85 :
      tempC < 800  ? 0.6 : 0.0,  // 0 = нельзя строить
  
    solarEfficiency:
      tempC < -100 ? 0.5 :
      tempC < -30  ? 0.7 :
      tempC < 10   ? 1.0 :
      tempC < 35   ? 1.1 :
      tempC < 60   ? 1.15 :
      tempC < 150  ? 1.2 :
      tempC < 800  ? 1.3 : 1.5,
    
    colonistHealth:
      tempC < -100 ? 0.6 :
      tempC < -30  ? 0.8 :
      tempC < 10   ? 0.95 :
      tempC < 35   ? 1.0 :
      tempC < 60   ? 0.9 :
      tempC < 150  ? 0.7 : 0.3,
  };
}
```

**Генерация температуры**: базовая температура зависит от расстояния до звезды и её светимости:
```
T_planet = T_star × sqrt(R_star × 0.00465 / (2 × D_AU)) − 273.15
```
где `T_star` — температура звезды (K), `R_star` — радиус звезды в R☉ (солнечных радиусах), `D_AU` — расстояние до звезды в астрономических единицах (AU), `0.00465` — коэффициент перевода R☉ → AU (1 R☉ = 0.00465 AU). К полученному значению прибавляется случайное отклонение ±20% с учётом парникового эффекта атмосферы.

> **Проверка (Земля)**: T_star = 5778 K, R_star = 1 R☉, D = 1 AU → T = 5778 × sqrt(0.00465 / 2) − 273 = 5778 × 0.0482 − 273 = 278.5 − 273 = **5.5 °C** ✓

## 3. Сетка застройки планеты

### 3.1 Общее описание

Застройка планеты организована в **три слоя**, каждый из которых предоставляет различные возможности для строительства:

1. **Поверхность** — гексагональная сетка на поверхности планеты (только для твёрдых планет)
2. **Верхняя атмосфера** — атмосферные платформы (только для газовых гигантов)
3. **Орбита** — орбитальные станции и верфи (для всех планет)

### 3.2 Слои застройки

#### 3.2.1 Поверхность (Surface hexes)

Слой поверхности доступен только для **твёрдых планет** (все типы, кроме газового гиганта). Представляет собой гексагональную сетку на поверхности планеты (см. §3.3). Каждый гекс может содержать одно наземное здание.

| Параметр | Значение |
|----------|----------|
| Доступность | Все типы, кроме газового гиганта |
| Количество слотов | Определяется классом размера планеты (19–127 гексов, см. §2.1) |
| Типы зданий | Все наземные здания (шахты, заводы, лаборатории, электростанции и т.д.) |

Газовые гиганты имеют **0 гексов поверхности** — застройка поверхности для них невозможна.

#### 3.2.2 Верхняя атмосфера (Upper atmosphere)

Слой верхней атмосферы доступен **только для газовых гигантов**. Представляет собой платформы, парящие в верхних слоях атмосферы и осуществляющие добычу атмосферных газов.

| Параметр | Значение |
|----------|----------|
| Доступность | Только газовые гиганты |
| Количество слотов | 6–12 (зависит от размера газового гиганта, определяется при генерации) |
| Типы зданий | Газовый экстрактор, Атмосферная платформа, Исследовательская станция (ограниченный набор) |

**Ограничения зданий верхней атмосферы**: в этом слое можно строить только здания, специально предназначенные для работы в атмосфере газового гиганта. Шахты, заводы, лаборатории общего назначения и другие наземные здания недоступны.

**Структура данных**:
```typescript
interface AtmosphericSlot {
  index: number;                         // 0..atmosphericSlots-1
  building: AtmosphericBuilding | null;  // здание или пусто
}

type AtmosphericBuildingType = 'gas_extractor' | 'atmospheric_platform' | 'research_station';

interface AtmosphericBuilding {
  type: AtmosphericBuildingType;
  level: number;  // 1–10
}
```

#### 3.2.3 Орбита (Orbit)

Орбитальный слой доступен **для всех планет**. На орбите строятся орбитальные станции, верфи, склады и другие космические сооружения. Подробное описание орбитальных станций — см. раздел 6.

| Параметр | Значение |
|----------|----------|
| Доступность | Все типы планет |
| Количество слотов (твёрдые планеты) | 3–6 (зависит от класса размера: Крошечная=3, Малая=4, Средняя=5, Большая=5, Огромная=6) |
| Количество слотов (газовые гиганты) | 6–12 (больше гравитационный колодец, выгоднее позиционирование) |
| Типы зданий | Орбитальные станции (верфь, торговая, оборонительная, солнечная ЭС, атмосферный зонд) |

**Структура данных**:
```typescript
function getOrbitSlots(planetType: PlanetType, sizeClass: PlanetSizeClass): number {
  if (planetType === 'gas_giant') {
    // 6–12 для газовых гигантов (зависит от размера)
    return 6 + Math.floor(Math.random() * 7);  // определяется при генерации
  }
  switch (sizeClass) {
    case 'tiny':   return 3;
    case 'small':  return 4;
    case 'medium': return 5;
    case 'large':  return 5;
    case 'huge':   return 6;
  }
}
```

### 3.3 Размер сетки поверхности

Размер сетки поверхности определяется классом размера планеты (см. раздел 2.1):

| Класс размера | Количество гексов |
|---------------|-------------------|
| Крошечная | 19 |
| Малая | 37 |
| Средняя | 61 |
| Большая | 91 |
| Огромная | 127 |
| Газовый гигант | 0 (застройка поверхности невозможна) |

Все размеры сетки образуют **полные гексагональные кольца** — количество гексов в сетке из n колец: `1 + 3 × n × (n + 1)`.

| Класс размера | Количество колец | Формула | Итого гексов |
|---------------|------------------|---------|-------------|
| Крошечная | 2 | 1 + 6 + 12 | 19 |
| Малая | 3 | 1 + 6 + 12 + 18 | 37 |
| Средняя | 4 | 1 + 6 + 12 + 18 + 24 | 61 |
| Большая | 5 | 1 + 6 + 12 + 18 + 24 + 30 | 91 |
| Огромная | 6 | 1 + 6 + 12 + 18 + 24 + 30 + 36 | 127 |

**Схема сеток для программиста**:

Крошечная (19 гексов, 2 кольца):
```
      [01] [02] [03]
   [06] [07] [08] [04]
[12] [13] [00] [09] [05]
   [11] [14] [10] [17]
      [16] [15] [18]
```

Малая (37 гексов, 3 кольца):
```
            [01] [02] [03] [04]
         [12] [13] [14] [15] [05]
      [11] [24] [25] [26] [16] [06]
   [10] [23] [30] [31] [27] [17] [07]
      [09] [22] [29] [28] [18] [36]
         [08] [21] [20] [19] [35]
            [34] [33] [32]
```

Средняя (61 гекс, 4 кольца):
```
               ...4 кольца, 61 гекс...
```

> **Примечание**: Точная конфигурация гексов определяется стандартной hex-grid системой с axial координатами (q, r). Сетка всегда состоит из полных колец — обрезка не требуется.

**Формула количества гексов в полном кольце**: `кольцо n` содержит `6 × n` гексов (для n ≥ 1). Центральный гекс = кольцо 0 (1 гекс). Суммарное количество гексов в сетке из N колец: `1 + 3 × N × (N + 1)`.

**Формула для программиста**:
```typescript
interface HexCoord {
  q: number;  // axial coordinate
  r: number;  // axial coordinate
}

function generateHexGrid(size: number): HexCoord[] {
  if (size === 0) return [];  // Газовый гигант
  
  // Определяем количество полных колец по размеру
  const ringsBySize: Record<number, number> = {
    19: 2, 37: 3, 61: 4, 91: 5, 127: 6
  };
  const maxRings = ringsBySize[size] ?? Math.floor((-1 + Math.sqrt(1 + 4 * (size - 1) / 3)) / 2);
  
  const hexes: HexCoord[] = [{ q: 0, r: 0 }];  // центральный гекс
  
  for (let ring = 1; ring <= maxRings; ring++) {
    let q = 0, r = -ring;
    const directions = [
      [1, 0], [1, 1], [0, 1], [-1, 0], [-1, -1], [0, -1]
    ];
    for (const [dq, dr] of directions) {
      for (let i = 0; i < ring; i++) {
        hexes.push({ q, r });
        q += dq;
        r += dr;
      }
    }
  }
  
  return hexes;  // Сетка из полных колец — обрезка не нужна
}
```

### 3.4 Типы местности

Каждый гекс поверхности имеет тип местности. Тип местности генерируется при создании планеты и определяет бонусы/штрафы для зданий, размещённых на данном гексе. Слой верхней атмосферы и орбитальный слой не имеют типов местности.

| Тип местности | Описание | Штраф к строительству | Бонус к добыче | Бонус к энергообеспечению | Бонус к исследованиям | Допустимые здания |
|---------------|----------|----------------------|----------------|--------------------------|----------------------|-------------------|
| **Равнина** | Плоский участок, пригодный для любого строительства | 0% | 0% | 0% | 0% | Все |
| **Горы** | Гористая местность, богатая минералами | +20% стоимость | +25% добыча | 0% | +10% (обсерватории) | Шахты, Лаборатории, Оборонительные платформы |
| **Пустыня** | Засушливая равнина | +5% стоимость | +10% добыча (редкие) | +15% (солнечные) | 0% | Шахты, Солнечные ЭС, Заводы |

**Алгоритм генерации для программиста**:
```
function generateTerrainHexes(
  planetType: PlanetType, 
  hexCount: number, 
  rng: SeededRandom
): TerrainType[] {
  const distribution = TERRAIN_DISTRIBUTION[planetType];
  const terrain: TerrainType[] = [];
  
  for (let i = 0; i < hexCount; i++) {
    terrain.push(rng.weightedChoice(distribution));
  }
  
  // Кластеризация: neighbouring гексы одного типа «притягиваются»
  // (1–2 прохода алгоритма сглаживания)
  for (let pass = 0; pass < 2; pass++) {
    for (let i = 0; i < terrain.length; i++) {
      const neighbors = getNeighborIndices(i, hexCount);
      const neighborTypes = neighbors.map(n => terrain[n]);
      // Если большинство соседей — другой тип, переключаемся с вероятностью 30%
      const mostCommon = mode(neighborTypes);
      if (mostCommon !== terrain[i] && rng.random() < 0.3) {
        terrain[i] = mostCommon;
      }
    }
  }
  
  return terrain;
}
```

## 4. Бонусы и штрафы планет

### 4.1 Сводная таблица всех модификаторов

Итоговая эффективность здания на планете рассчитывается как произведение всех модификаторов:

```
итоговая_эффективность = базовая_эффективность 
  × gravity_modifier 
  × temperature_modifier 
  × atmosphere_modifier 
  × terrain_modifier 
  × magnetic_field_modifier
```

```
итоговая_стоимость = базовая_стоимость 
  × gravity_cost_modifier 
  × atmosphere_cost_modifier 
  × terrain_cost_modifier
```

### 4.2 Модификатор гравитации

| Параметр | Формула |
|----------|---------|
| Стоимость строительства | `baseCost × (g < 0.5 ? 0.8 : g < 1.2 ? 1.0 : g < 2.0 ? 1.2 : 1.4)` |
| Скорость строительства | `buildTime × (g < 0.5 ? 0.75 : g < 1.2 ? 1.0 : g < 2.0 ? 1.15 : 1.3)` |
| Скорость добычи | `miningRate × (g < 0.5 ? 0.85 : g < 1.2 ? 1.0 : g < 2.0 ? 1.15 : 1.3)` |
| Здоровье колонистов | `health × (g < 0.05 ? 0.5 : g < 0.2 ? 0.8 : g < 2.0 ? 1.0 : 0.7)` |

**Обоснование**: 
- Низкая гравитация — дешевле выводить груз на орбиту и возводить конструкции (меньше нагрузка), но тяжёлая техника работает менее эффективно (меньше сцепление, медленнее падение в шахтах)
- Высокая гравитация — дороже строить (усиленные конструкции), но добыча идёт быстрее (тяжёлые буровые установки эффективнее)

### 4.3 Модификатор атмосферы

| Тип атмосферы | Модификатор стоимости зданий | Модификатор эффективности | Примечание |
|---------------|------------------------------|--------------------------|------------|
| Нет | ×0.9 | ×1.0 | Автоматизация дешевле (не нужна вентиляция), но все здания — герметичные |
| Тонкая | ×1.05 | ×0.95 | Частичная герметизация |
| Стандартная | ×1.0 | ×1.0 | Без штрафов |
| Плотная | ×1.1 | ×0.95 | Усиленные конструкции из-за давления |
| Токсичная | ×1.3 | ×0.85 | Полная герметизация + фильтрация + коррозионная защита |
| Инертная | ×1.1 | ×0.95 | Герметизация (нет O₂) |
| Метановая | ×1.2 | ×0.9 | Герметизация + взрывозащита |
| CO₂ | ×1.15 | ×0.9 | Герметизация (кислород не добывается из атмосферы) |

**Дополнительное влияние атмосферы**:
- **Нет атмосферы**: Автоматизированные здания (без экипажа) получают бонус −10% к стоимости (не нужны системы жизнеобеспечения)
- **Стандартная**: Здания с экипажем получают +5% к эффективности (комфортные условия)
- **Токсичная**: Если на планете построена лаборатория, появляется шанс (1% за тик) исследования новых химических технологий из анализа токсичных соединений
- **Метановая**: Возможна добыча CH₄ — ценного промышленного сырья

### 4.4 Модификатор температуры

Подробная таблица — см. раздел 2.3.

**Дополнительные эффекты**:
- Экстремальный холод (−100 °C): здания требуют **подогрева** — дополнительное потребление энергии (+20% к энергопотреблению)

### 4.5 Модификатор наличия жизни

| Уровень жизни | Бонус к габитабельности | Бонус к био-исследованиям | Шанс случайного открытия | Бонус к производству пищи |
|---------------|------------------------|--------------------------|-------------------------|--------------------------|
| Нет | 0% | ×1.0 | 0% | ×0.5 (только синтетическая пища) |

| Растения | +10% | ×1.2 | 3%/тик | ×1.0 |

**Бонус к производству пищи**: на планетах с жизнью возможно сельское хозяйство и аквакультура, значительно снижающие зависимость от импорта продовольствия.

**Механика «шанс случайного открытия»**: при наличии лаборатории с назначенным исследованием, каждый тик рассчитывается шанс. При срабатывании — текущее исследование получает прогресс +5–20%. Это не даёт бесплатных технологий, а ускоряет текущую работу.

|-----------|-----------------|---------------------|------------------------------|
| Мощное (>2.0) | +10% | 1.0 | 0%/год |
| Стандартное (0.5–2.0) | 0% | 1.0 | 2%/год |
| Слабое (0.1–0.5) | −5% | 0.9 | 8%/год |
| Отсутствует (0) | −15% | 0.75 | 20%/год |

**Деградация солнечных панелей**: без магнитного поля солнечные панели деградируют быстрее, теряя эффективность. Это моделируется как уменьшение максимальной эффективности на указанный процент за игровой год. Требует периодической замены панелей (расход Si).

### 4.7 Расчёт габитабельности

Габитабельность — сводный показатель комфортности планеты для колонистов. Влияет на скорость роста населения и эффективность работы.

```
habitability = base_habitability_by_type
  + atmosphere_habitability_bonus
  + temperature_habitability_factor
  + life_habitability_bonus
  + magnetic_field_habitability_bonus
  − gravity_habitability_penalty
```

Все слагаемые — в процентах (0–100). Результат ограничивается диапазоном 0–100.

| Габитабельность | Рост населения | Эффективность колонистов |
|-----------------|----------------|--------------------------|
| 0–10% | −0.5% / год (убыль) | ×0.3 |
| 10–30% | +0.1% / год | ×0.5 |
| 30–50% | +0.5% / год | ×0.7 |
| 50–70% | +1.0% / год | ×0.85 |
| 70–90% | +1.5% / год | ×0.95 |
| 90–100% | +2.0% / год | ×1.0 |

**Формула для программиста**:
```typescript
function calculateHabitability(planet: Planet): number {
  let h = PLANET_BASE_HABITABILITY[planet.type];
  
  // Атмосфера
  h += ATMOSPHERE_HABITABILITY_BONUS[planet.atmosphere.type];
  
  // Температура (оптимум 10–35°C)
  const tempDelta = Math.abs(planet.temperature - 22); // 22°C = оптимум
  h -= tempDelta > 200 ? 40 : tempDelta > 100 ? 25 : tempDelta > 50 ? 15 : tempDelta > 13 ? 5 : 0;
  
  // Жизнь
  h += LIFE_HABITABILITY_BONUS[planet.life.level];
  
  // Магнитное поле
  if (planet.magneticField > 2.0) h += 10;
  else if (planet.magneticField < 0.1) h -= 15;
  else if (planet.magneticField < 0.5) h -= 5;
  
  // Гравитация (оптимум 0.8–1.2g)
  const gDelta = Math.abs(planet.gravity - 1.0);
  h -= gDelta > 1.5 ? 25 : gDelta > 0.7 ? 15 : gDelta > 0.3 ? 5 : 0;
  
  return Math.max(0, Math.min(100, h));
}
```

## 5. Ресурсный потенциал планеты

### 5.1 Философия ресурсов

**Ключевой принцип**: каждая планета содержит **ВСЕ 22 элемента** из игровой «таблицы Менделеева» (MVP). Разница между планетами — не в наличии/отсутствии элементов, а в **количестве** и **доступности**.

Это обеспечивает три важных игровых свойства:

1. **Любая планета — стартовая**: игрок может начать базовое производство на любой планете, поскольку редкие элементы (W, Co, Pt, Y, Ba, Au, U) присутствуют везде хотя бы в следовых количествах.
2. **Специализация по типу**: каждый тип планеты имеет «профильные» элементы в значительно больших количествах, что создаёт экономическую специализацию.
3. **Уникальные находки**: 1–2 ультраредких элемента на планете делают каждый мир потенциально ценным — именно ради них игроки исследуют далёкие системы.

### 5.2 Три уровня ресурсов (tiers)

Все 22 элемента MVP распределяются по трём уровням:

#### 5.2.1 Профильные (Profile)

Элементы, характерные для данного типа планеты. Присутствуют в **значительно больших количествах** — множитель 3.0–5.0× от базового.

| Тип планеты | Профильные элементы | Обоснование |
|-------------|---------------------|-------------|
| Скалистая | Fe, Si, Al, C, O | Стандартный набор для строительства, богатство недр |
| Вулканическая | Fe, Ti, Cr, Ni, S, V | Тяжёлые металлы, вулканические выбросы, тектоническая активность |
| Ледяная | H, O, N, He | Лёткие элементы, замёрзшие газы, ледяные шапки |
| Океаническая | H, O, N, C, S | Морские ресурсы, растворённые газы, органика |
| Пустынная | Si, Fe, Al, Ti | Эрозия обнажает минеральные залежи, пустынные россыпи |
| Газовый гигант | H, He, C, N | Атмосферные газы, основной состав газовых гигантов |
| Карликовая | Fe, Si, C, Ni | Базовые элементы, часто астероидного происхождения |

**Характеристики профильных залежей**:

| Параметр | Значение |
|----------|----------|
| Множитель количества | 3.0 – 5.0× базового |
| Доступность (avgAvailability) | 0.30 – 0.80 |
| Количество (totalQuantity) | 30 000 – 500 000 тыс. тонн |
| Количество гексов (hexCount) | Многие гексы содержат залежи |
| Описание | Промышленные залежи, основа экономики планеты |

#### 5.2.2 Редкие (Rare)

Определённый набор элементов, присутствующих на **каждой планете** в следовых количествах. Множитель 0.1–0.3× от базового. Всегда доступны, чтобы игрок мог запустить базовое производство.

**Список редких элементов (постоянный для всех планет)**: W (Вольфрам), Co (Кобальт), Pt (Платина), Y (Иттрий), Ba (Барий), Au (Золото), U (Уран).

> **Дизайн-обоснование**: эти 7 элементов критически важны для высокотехнологичных цепочек (сверхпроводник, микрочипы, ядерная энергия). Их гарантированное присутствие на каждой планете означает, что игрок никогда не окажется полностью заблокированным — он всегда может начать производство, пусть и медленно.

**Характеристики редких залежей**:

| Параметр | Значение |
|----------|----------|
| Множитель количества | 0.1 – 0.3× базового |
| Доступность (avgAvailability) | 0.05 – 0.20 |
| Количество (totalQuantity) | 500 – 5 000 тыс. тонн |
| Количество гексов (hexCount) | Немногие гексы содержат залежи |
| Описание | Следовые количества, добыча рентабельна при наличии спроса |

#### 5.2.3 Ультраредкие (Ultra-rare)

1–2 элемента, уникальных для конкретной планеты. Минимальные количества, крайне низкая доступность. Это «находки» — ценные открытия, ради которых стоит лететь в дальние системы.

**Пул ультраредких элементов**: W, Co, Pt, Y, Ba, Au, U, Li, V.

При генерации планеты из пула случайным образом выбираются 1–2 элемента. Они получают статус ультраредких с минимальными параметрами.

**Характеристики ультраредких залежей**:

| Параметр | Значение |
|----------|----------|
| Множитель количества | 0.02 – 0.05× базового |
| Доступность (avgAvailability) | 0.01 – 0.05 |
| Количество (totalQuantity) | 1 – 6 тыс. тонн |
| Количество гексов (hexCount) | 1–2 гекса содержат залежи |
| maxAvailability | 0.03 – 0.05 |
| Описание | Уникальные находки, добыча крайне медленная, но ценность высока |

> **Дизайн-обоснование**: ультраредкие элементы создают «алчность исследования» — игрок хочет обследовать каждую планету в поисках уникальных ресурсов. Это мотивирует экспансию и разведку.

### 5.3 Структура данных

#### 5.3.1 Сводная залежь планеты (PlanetResourceDeposit)

Агрегирует все залежи одного элемента со всех гексов планеты. Хранится в `planet.resourceDeposits[]`.

```typescript
interface PlanetResourceDeposit {
  elementId: string;          // ID элемента (напр. "Fe", "Au", "Y")
  totalQuantity: number;      // Суммарное количество ресурса (тыс. тонн)
  avgAvailability: number;    // Средняя доступность (0–1)
  tier: 'profile' | 'rare' | 'ultra_rare';  // Категория залежи
  hexCount: number;           // Количество гексов, где ресурс присутствует
  maxAvailability: number;    // Максимальная доступность среди всех залежей элемента
}
```

**Пояснение полей**:

| Поле | Описание | Влияние на геймплей |
|------|----------|---------------------|
| `elementId` | Идентификатор элемента из таблицы Менделеева | Определяет, что именно добывается |
| `totalQuantity` | Суммарный запас (тыс. тонн) по всем гексам | Определяет, как долго залежь будет исчерпана |
| `avgAvailability` | Средняя лёгкость добычи (0–1) | Влияет на скорость: `добыча = базовая_скорость × avgAvailability` |
| `tier` | Категория: profile / rare / ultra_rare | Определяет визуальное отображение и стратегическую ценность |
| `hexCount` | Сколько гексов содержат этот ресурс | Показывает распространённость залежи на планете |
| `maxAvailability` | Пиковая доступность среди гексов | Игрок может выбрать лучший гекс для максимальной добычи |

#### 5.3.2 Залежь в гексе (ResourceDeposit)

Каждый гекс поверхности может содержать локальные залежи. Хранится в `hexCell.deposits[]`.

```typescript
interface ResourceDeposit {
  elementId: string;       // ID элемента (напр. "Fe-ore", "Au-ore")
  availability: number;    // 0.01 – 1.0 (скорость добычи)
  quantity: number;        // оставшееся количество (тыс. тонн)
  depth: number;           // 1 – 5 (глубина залегания, влияет на стоимость шахты)
}
```

> **Примечание**: `elementId` в гексовой залежи содержит ID руды (напр. `"Fe-ore"`), а не чистого элемента. Руда перерабатывается в чистый элемент через плавильню (см. 04-buildings.md).

```typescript
  dwarf:    ['Fe', 'Si', 'C', 'Ni'],
};

/** Редкие элементы — на КАЖДОЙ планете, множитель 0.1–0.3x */
export const RARE_ELEMENTS = ['W', 'Co', 'Pt', 'Y', 'Ba', 'Au', 'U'];

/** Ультраредкие элементы — пул для выбора 1-2 на планету, множитель 0.02–0.05x */
export const ULTRA_RARE_ELEMENTS = ['W', 'Co', 'Pt', 'Y', 'Ba', 'Au', 'U', 'Li', 'V'];
```

### 5.4 Доступность и скорость добычи

**Доступность** (availability / avgAvailability) определяет скорость добычи элемента:

| Доступность | Скорость добычи | Описание | Типичный tier |
|-------------|-----------------|----------|---------------|
| 0.01 – 0.05 | Очень медленно | Ультраредкие находки, нужны шахты уровня 8+ | ultra_rare |
| 0.05 – 0.15 | Медленно | Редкие элементы, рентабельно при наличии спроса | rare |
| 0.15 – 0.30 | Ниже среднего | Умеренные залежи | rare / profile (худший случай) |
| 0.30 – 0.50 | Средне | Стандартные промышленные залежи | profile |
| 0.50 – 0.80 | Выше среднего | Богатые залежи, основа экономики | profile |
| 0.80 – 1.00 | Очень богато | Исключительно богатые залежи | profile (лучший случай) |

**Формула скорости добычи**:
```
добыча_за_тик = базовая_скорость_шахты × availability × gravity_mining_modifier × terrain_mining_modifier
```

### 5.5 Количество и исчерпаемость

| Категория | Диапазон (тыс. тонн) | Tier | Время исчерпания |
|-----------|---------------------|------|------------------|
| Ультраредкая | 1 – 6 | ultra_rare | ~2–5 игровых лет |
| Редкая | 500 – 5 000 | rare | ~50–100 игровых лет |
| Профильная (малая) | 30 000 – 100 000 | profile | ~200 игровых лет |
| Профильная (крупная) | 100 000 – 500 000 | profile | ~500–2000 игровых лет |

**При исчерпании**: залежь помечается как `depleted`. Добыча прекращается. Здание (шахта) можно перепрофилировать на другую залежь (если доступна) или снести.

### 5.6 Генерация ресурсов

Генерация ресурсов происходит при создании планеты и состоит из трёх этапов:

#### Этап 1: Назначение tier для каждого элемента

Все 22 элемента MVP распределяются по трём категориям:

1. **Профильные**: элементы из `PROFILE_ELEMENTS[planetType]`
2. **Редкие**: элементы из `RARE_ELEMENTS` (7 элементов), если они ещё не назначены профильными
3. **Ультраредкие**: 1–2 случайных элемента из `ULTRA_RARE_ELEMENTS`, если они ещё не назначены профильными или редкими

> **Важно**: элемент может быть профильным на одном типе планет и редким на другом. Например, W (Вольфрам) — редкий элемент на скалистой планете, но может быть профильным на вулканической.

#### Этап 2: Генерация параметров по tier

```typescript
function generatePlanetResourceDeposit(
  elementId: string,
  tier: 'profile' | 'rare' | 'ultra_rare',
  planetType: PlanetType,
  starType: StarType,
  rng: SeededRandom
): PlanetResourceDeposit {
  // Базовое количество из справочника
  const baseQuantity = ELEMENT_BASE_QUANTITY[elementId];

  // Множитель по tier
  const tierMultiplier = tier === 'profile'
    ? rng.range(3.0, 5.0)
    : tier === 'rare'
    ? rng.range(0.1, 0.3)
    : rng.range(0.02, 0.05);

  // Модификатор типа звезды
  const starMod = STAR_RESOURCE_MODIFIER[starType]?.[elementId] ?? 1.0;

  const totalQuantity = Math.floor(baseQuantity * tierMultiplier * starMod * rng.range(0.7, 1.3));

  // Доступность по tier
  const avgAvailability = tier === 'profile'
    ? rng.range(0.30, 0.80)
    : tier === 'rare'
    ? rng.range(0.05, 0.20)
    : rng.range(0.01, 0.05);

  // Количество гексов с залежами
  const hexCount = tier === 'profile'
    ? Math.floor(rng.range(0.3, 0.7) * totalHexCount)   // 30–70% гексов
    : tier === 'rare'
    ? Math.floor(rng.range(0.05, 0.15) * totalHexCount) // 5–15% гексов
    : rng.range(1, 3);                                    // 1–2 гекса

  return {
    elementId,
    totalQuantity,
    avgAvailability,
    tier,
    hexCount,
    maxAvailability: avgAvailability * rng.range(1.2, 1.5),  // Лучший гекс чуть выше среднего
  };
}
```

#### Этап 3: Влияние типа звезды

Тип звезды модифицирует количество ресурсов (на этапе генерации):

| Тип звезды | Влияние на ресурсы |
|------------|-------------------|
| Жёлтый карлик (STAR_G) | Базовые значения (модификатор 1.0) |
| Красный карлик (STAR_M) | ×0.8 количество тяжёлых элементов, ×1.1 лёгких |
| Белый карлик (STAR_WD) | ×1.5 доступность тяжёлых металлов, ×0.7 лёгких |
| Красный гигант (STAR_RG) | ×1.2 количество всех ресурсов |
| Голубой гигант (STAR_O/B) | ×1.3 редкие элементы, ×0.8 базовые |
| Нейтронная звезда (STAR_NS) | ×2.0 ультраредкие, ×0.5 остальные |

## 6. Орбитальные станции

### 6.1 Общее описание

Орбитальные станции строятся на орбите вокруг планет и астероидных полей. Они не зависят от поверхности планеты и могут быть построены вокруг любого небесного тела. Орбитальные станции занимают слоты орбитального слоя застройки (см. §3.2.3).

**Ограничения орбитальных слотов**:
- **Крошечная** планета: 3 орбитальных слота
- **Малая** планета: 4 орбитальных слота
- **Средняя** планета: 5 орбитальных слотов
- **Большая** планета: 5 орбитальных слотов
- **Огромная** планета: 6 орбитальных слотов
- **Газовый гигант**: 6–12 орбитальных слотов (больше гравитационный колодец, выгоднее позиционирование)
- **Астероидное поле**: 2 орбитальных слота
- Каждая станция занимает один «орбитальный слот»

#### 6.2.4 Солнечная электростанция (Solar Power Station)

Орбитальная солнечная электростанция. Эффективность зависит от расстояния до звезды и её светимости.

| Параметр | Значение |
|----------|----------|
| Стоимость строительства | Стальной сплав(20) + Сверхпроводник(5) + Энергия(100) |
| Энергопотребление | 0 (станция производит энергию) |
| Выработка энергии | Зависит от расстояния до звезды (см. формулу) |
| Уровни | 1–5 (каждый уровень +50% выработки) |

| Уровень | Выработка (базовая) | Стоимость улучшения |
|---------|---------------------|---------------------|
| 1 | 100 МВт/тик | — |
| 2 | 150 МВт/тик | Стальной сплав(15) + Сверхпроводник(3) + Энергия(80) |
| 3 | 200 МВт/тик | Стальной сплав(25) + Сверхпроводник(6) + Энергия(150) |
| 4 | 250 МВт/тик | Стальной сплав(40) + Сверхпроводник(10) + Энергия(300) |
| 5 | 300 МВт/тик | Стальной сплав(60) + Сверхпроводник(15) + Энергия(500) |

**Формула выработки**:
```
выработка = базовая_выработка × уровень_множитель × star_luminosity / (distance_AU²)
```

где:
- `базовая_выработка` = 100 МВт/тик
- `уровень_множитель` = 1.0 + (level − 1) × 0.5
- `star_luminosity` = светимость звезды в солнечных единицах (1.0 для G-типа)
- `distance_AU` = расстояние от станции до звезды в астрономических единицах

**Примеры**:
- Орбита Земли (1 АЕ, светимость 1.0): 100 МВт/тик (базовый уровень 1)
- Орбита Марса (1.5 АЕ, светимость 1.0): 100 / 1.5² = 44 МВт/тик
- Орбита Меркурия (0.4 АЕ, светимость 1.0): 100 / 0.4² = 625 МВт/тик
- Голубой гигант (светимость 100, 5 АЕ): 100 × 100 / 25 = 400 МВт/тик

#### 6.2.5 Атмосферный зонд (Atmospheric Harvester) — только для газовых гигантов

Специализированная станция для добычи лёгких элементов из атмосферы газового гиганта.

| Параметр | Значение |
|----------|----------|
| Стоимость строительства | Стальной сплав(35) + Титановый сплав(10) + Сверхпроводник(5) + Энергия(400) |
| Энергопотребление | 40 МВт/тик |
| Добыча | Зависит от состава атмосферы и уровня |
| Ограничение | Только для газовых гигантов |

| Уровень | Скорость добычи | Типы добываемых элементов | Стоимость улучшения |
|---------|----------------|--------------------------|---------------------|
| 1 | ×1.0 | H, He | — |
| 2 | ×1.3 | H, He, C | Стальной сплав(20) + Энергия(200) |
| 3 | ×1.6 | H, He, C, N | Стальной сплав(35) + Сверхпроводник(5) + Энергия(400) |
| 4 | ×2.0 | H, He, C, N, CH₄ | Титановый сплав(15) + Сверхпроводник(10) + Энергия(800) |
| 5 | ×2.5 | H, He, C, N, CH₄, NH₃, He-3 | Сверхпроводник(20) + Микрочип(10) + Энергия(1500) |

**Скорость добычи**:
```
добыча_элемента_за_тик = базовая_скорость × уровень_множитель × процент_в_атмосфере × 10
```

где `базовая_скорость` = 5 тонн/тик, `процент_в_атмосфере` — доля элемента в составе атмосферы (0.0–1.0).

### 7.1 Полная структура данных планеты

### 7.3 Алгоритм генерации планеты (псевдокод)

```typescript
function generatePlanet(
  starType: StarType,
  orbitalDistanceAU: number,
  rng: SeededRandom
): Planet {
  // 1. Определить тип планеты на основе расстояния и звезды
  const type = determinePlanetType(starType, orbitalDistanceAU, rng);
  
  // 2. Сгенерировать радиус
  const radius = rng.range(RADIUS_RANGE[type].min, RADIUS_RANGE[type].max);
  
  // 3. Вычислить плотность и гравитацию
  const density = rng.range(DENSITY_RANGE[type].min, DENSITY_RANGE[type].max);
  const earthRadius = 6371;
  const earthDensity = 5.51;
  const gravity = Math.pow(radius / earthRadius, 2) * (density / earthDensity);
  
  // 4. Вычислить температуру
  const baseTempK = calculateEquilibriumTemperature(starType, orbitalDistanceAU);
  const greenhouseFactor = getGreenhouseFactor(type);
  const temperature = baseTempK * greenhouseFactor - 273.15;
  
  // 5. Определить класс размера и сетку
  const radiusInEarthRadii = radius / 6371;
  const sizeClass = determineSizeClass(type, radiusInEarthRadii);
  const gridSize = GRID_SIZES[sizeClass];
  
  // 6. Сгенерировать атмосферу
  const atmosphere = generateAtmosphere(type, temperature, gravity, rng);
  
  // 7. Определить магнитное поле
  const magneticField = generateMagneticField(type, radius, density, rng);
  
  // 8. Определить наличие жизни
  const life = generateLife(type, temperature, atmosphere, magneticField, rng);
  
  // 9. Генерация сетки (только для твёрдых планет)
  const hexGrid = type === 'gas_giant' ? [] : generateHexGrid(gridSize);
  if (hexGrid.length > 0) {
    const terrainTypes = generateTerrainHexes(type, gridSize, rng);
    for (let i = 0; i < hexGrid.length; i++) {
      hexGrid[i].terrain = terrainTypes[i];
    }
  }
  
  // 10. Генерация ресурсов
  const deposits = generateDeposits(type, starType, rng);
  assignDepositsToHexes(deposits, hexGrid, rng);
  
  // 11. Вычислить габитабельность
  const habitability = calculateHabitability(/* все параметры */);
  
  // 12. Спутники (для больших планет и газовых гигантов)
  const moons = generateMoons(type, radius, starType, rng);
  
  return {
    id: generateId(),
    name: generateName(rng),
    type,
    radius,
    gravity,
    temperature,
    density,
    atmosphere,
    magneticField,
    life,
    hasComplexOrganics: determineOrganics(type, life, rng),
    organicQuantity: /* ... */,
    organicAvailability: /* ... */,
    sizeClass,
    hexGrid,
    atmosphericSlots: type === 'gas_giant' ? generateAtmosphericSlots(rng) : [],
    orbitSlots: getOrbitSlots(type, sizeClass),
    deposits,
    orbitalStations: [],
    moons,
    orbitalDistanceAU,
    ownerId: null,
    colonyId: null,
    habitability,
  };
}
```

### 7.4 Правила генерации типа планеты по орбитальной зоне

| Зона (АЕ от звезды G-типа) | Вероятности типов планет |
|----------------------------|-------------------------|
| 0.1 – 0.5 (ближняя) | Карликовая (20%), Скалистая (15%), Вулканическая (50%), Пустынная (15%) |
| 0.5 – 1.5 (внутренняя) | Карликовая (10%), Скалистая (35%), Вулканическая (15%), Пустынная (25%), Океаническая (15%) |
| 1.5 – 3.0 (обитаемая) | Скалистая (25%), Океаническая (30%), Пустынная (20%), Ледяная (15%), Карликовая (10%) |
| 3.0 – 10.0 (внешняя) | Ледяная (35%), Газовый гигант (30%), Скалистая (15%), Карликовая (20%) |
| 10.0+ (дальняя) | Газовый гигант (50%), Ледяная (30%), Карликовая (20%) |

> **Для других типов звёзд**: зоны масштабируются пропорционально светимости. Формула: `outer_bound = inner_bound × sqrt(luminosity / sun_luminosity)`.
