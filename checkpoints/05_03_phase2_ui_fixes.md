**Дата:** 2026-05-03 19:30 MSK

## Выполненные задачи

### UI-исправления (сессия 1)
1. Исправлена ошибка React hydration — фон звёзд/частиц использовал Math.random(), вызывая SSR-рассинхронизацию. Заменён на детерминированный PRNG seededRng().
2. Исправлена блокировка cross-origin в Next.js 16 с превью-домена *.space-z.ai — добавлен allowedDevOrigins в next.config.ts.
3. Реализован зум на карте галактики через колесо мыши с центрированием на курсоре.
4. Увеличен максимальный зум с 15x до 80x для детального изучения систем.
5. Добавлен компонент мини-карты, появляющийся при зуме > 3x.
6. Добавлены кнопки зума +/− и кнопка Fit (сброс).
7. Зум по клику на звёздную систему (4x с центрированием на системе).
8. Контр-масштабирование размеров точек и шрифтов для читаемости при любом зуме.

### Исправления движка и рецептов (сессия 1)
9. Исправлены 14 рецептов плавки (P2-03 / Р3-01) — рудные месторождения теперь производят руды, а не чистые элементы.
10. Исправлены 6 критических багов движка:
    - P2-06/P2-07: выход ядерной электростанции
    - P3-02: энергия за тик (per-tick energy)
    - P3-04: валидация размера здания
    - P3-06: проверка существования здания
    - P2-26: guard светимости чёрной дыры
    - P3-05: масштабирование скорости экономики

### Звёзды и навигация (сессия 2)
11. Цвета звёзд приведены к астрономическим спектральным классам (O→голубой, B→бело-голубой, A→белый, F→жёлто-белый, G→золотисто-жёлтый, K→оранжевый, M→красно-оранжевый).
12. Спецэффекты для экзотических звёзд: аккреционный диск у чёрных дыр, луч у пульсаров.
13. Исправлен сброс зума при возврате на карту галактики — GalaxyMap не размонтируется, а скрывается через CSS hidden.
14. Добавлена легенда спектральных классов (O B A F G K M) на карту галактики.
