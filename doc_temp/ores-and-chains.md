## 1. Обзор

### 1.1 Концепция цепочки трансформации

Вся экономика SpaceGame построена на многоуровневых цепочках переработки:

```
Руда / Сырьё  →  Здание переработки  →  Чистый элемент  →  Сплав / Материал  →  Компонент  →  Модуль  →  Корабль
```

**Ключевой принцип:** добывающие здания (Шахта, Карьер, Газовый экстрактор, Бурильная установка, Ледодобывающая станция) производят **руды и сырьевые соединения**, а не чистые элементы. Чистые элементы получаются только после переработки на Плавильне, Химическом заводе или Очистительном комплексе.

> **Критическое замечание (из аудита Р3-01):** текущий код `processExtraction` добавляет добытое напрямую как чистый элемент (`planet.resources['Fe']`), минуя стадию руды. Это разрывает цепочку крафта — рецепты переплавки требуют `'Fe-ore'`, который никогда не производится. Все руды в данном документе определены как промежуточные продукты, которые **должны** производиться добывающими зданиями.

### 1.2 Уровни переработки

| Уровень | Что происходит | Здание | Пример |
|---------|---------------|--------|--------|
| 0 — Добыча | Извлечение руды/сырья из природных источников | Шахта, Карьер, Газ. экстрактор, Бурильная уст., Ледодоб. станция | `Fe-ore` (железная руда) |
| 1 — Первичная переработка | Руда → чистый элемент; соединение → элементы | Плавильня, Химзавод | `Fe` (чистое железо) |
| 2 — Синтез материалов | Элементы → сплавы и материалы | Плавильня, Химзавод, Нефтехимзавод | `steel` (стальной сплав) |
| 3 — Производство компонентов | Материалы + элементы → компоненты | Завод электроники, Завод двигателей, Завод корпусов | `microchip` (микрочип) |
| 4 — Сборка модулей | Компоненты → модули кораблей | Верфь | `ion_engine` (ионный двигатель) |
| 5 — Сборка кораблей | Модули → корабль | Верфь | `scout` (разведчик) |

### 1.3 Обозначения

- **Руда** — природное минеральное сырьё, содержащее элемент в виде соединения. ID формата `{Символ}-ore` (напр. `Fe-ore`).
- **Соединение** — природное химическое соединение (газ, лёд, раствор). ID формата `{формула}` (напр. `H2O`, `CO2`).
- **Чистый элемент** — результат переработки. ID = символ элемента (напр. `Fe`, `Si`).
- **Коэффициент выхода** — сколько единиц чистого элемента получается из 10 единиц руды.

### 2.4 Чистые/самородные элементы

Некоторые элементы могут быть найдены в природе в **самородном (чистом) виде** — без необходимости переплавки из руды. В игре это означает, что добывающее здание может производить как руду (с последующей переплавкой), так и самородный элемент (напрямую, но реже).

| Элемент | Самородная форма | Где встречается | Вероятность vs руда | Примечание |
|---------|-----------------|-----------------|---------------------|------------|
| S | Самородная сера | Вулканические фумаролы | 30% — остальное `S-ore` | Включено в `S-ore` (пирит + самородная) |
| C | Графит, алмаз | Метаморфические породы | 20% — остальное `C-ore` | Включено в `C-ore` |
| Cu | Самородная медь | Гидротермальные жилы | 5% — остальное `Cu-ore` | Редкая находка, бонус к добыче |
| Ag | Самородное серебро | Гидротермальные жилы | 5% — остальное `Ag-ore` | Может добываться без переплавки |
| Au | Самородное золото | Россыпи, кварцевые жилы | 10% — остальное `Au-ore` | Включено в `Au-ore` |
| Pt | Самородная платина | Россыпи | 3% — остальное `Pt-ore` | Включено в `Pt-ore` |

> **Игровая механика:** при добыче `X-ore` на шахте/карьере есть шанс (указан в таблице) дополнительно получить чистый элемент X напрямую, минуя переплавку. Это моделирует находку самородков. Шанс повышается с уровнем здания добычи (+2% за уровень).

## 3. Атмосферные соединения

Газовый экстрактор извлекает из атмосферы не чистые элементы, а **газовые соединения**. Большинство газов используются напрямую, некоторые требуют переработки на Химзаводе.

### 3.1 Таблица атмосферных соединений

| ID соединения | Название | Формула | Компоненты | Тип атмосферы | Выход из 10 ед. | Здание переработки |
|---------------|----------|---------|------------|---------------|-----------------|-------------------|
| `H2` | Водород | H₂ | H₂ (чистый) | Плотная, Газ. гигант | 10 H₂ | — (используется напрямую как H) |
| `He` | Гелий | He | He (чистый) | Инертная, Газ. гигант | 10 He | — (используется напрямую) |
| `Ne` | Неон | Ne | Ne (чистый) | Инертная, Газ. гигант | 10 Ne | — (используется напрямую) |
| `Ar` | Аргон | Ar | Ar (чистый) | Инертная, Стандартная | 10 Ar | — (используется напрямую) |
| `N2` | Азот | N₂ | N₂ (чистый) | Тонкая, Стандартная, Плотная | 10 N₂ | — (используется напрямую как N) |
| `CO2` | Углекислый газ | CO₂ | C + O₂ | Тонкая, Плотная, Токсичная, CO₂ | 5 C + 9 O₂ | Химзавод |
| `CH4` | Метан | CH₄ | C + H₂ | Метановая, Газ. гигант | 2 C + 6 H₂ | Химзавод |
| `NH3` | Аммиак | NH₃ | N + H₂ | Токсичная, Метановая | 8 N + 12 H₂ | Химзавод |
| `H2S` | Сероводород | H₂S | H₂ + S | Токсичная | 6 H₂ + 8 S | Химзавод |
| `SO2` | Диоксид серы | SO₂ | S + O₂ | Токсичная, Вулканическая | 6 S + 8 O₂ | Химзавод |
| `O2` | Кислород | O₂ | O₂ (чистый) | Стандартная, Плотная | 10 O₂ | — (используется напрямую как O) |

> **Новое в версии 2.0:** добавлены инертные газы `Ne` (неон) и `Ar` (аргон). Ne добывается из инертной атмосферы и атмосферы газовых гигантов, используется для лазеров и криогеники. Ar добывается из инертной и стандартной атмосферы, используется как инертная среда для сварки и производства электроники. Оба газа не требуют переработки.

### 3.3 Цепочки переработки атмосферных газов

```
CO₂ (10)  ──[Химзавод]──▶  C (5) + O₂ (9)
CH₄ (10)  ──[Химзавод]──▶  C (2) + H₂ (6)
NH₃ (10)  ──[Химзавод]──▶  N (8) + H₂ (12)
H₂S (10)  ──[Химзавод]──▶  H₂ (6) + S (8)
SO₂ (10)  ──[Химзавод]──▶  S (6) + O₂ (8)
H₂O (10)  ──[Химзавод]──▶  H₂ (8) + O₂ (8)   ← из ледодобывающей станции
```

> **Для чистых газов (H₂, He, Ne, Ar, N₂, O₂):** газовый экстрактор производит их в готовом виде, переработка не нужна. В игровом балансе `H₂` → `H` (водород), `N₂` → `N` (азот), `O₂` → `O` (кислород) — преобразование автоматическое (1:1). Ne и Ar используются как самостоятельные элементы (не разлагаются).

## 4. Ледяные соединения

Ледодобывающая станция извлекает замёрзшие летучие вещества. Все продукты требуют переработки на Химзаводе.

### 4.1 Таблица ледяных соединений

| ID соединения | Название | Формула | Компоненты | Условия добычи | Выход из 10 ед. | Здание переработки |
|---------------|----------|---------|------------|----------------|-----------------|-------------------|
| `H2O-ice` | Водяной лёд | H₂O | H₂ + O₂ | T < +50°C | 8 H₂ + 8 O₂ | Химзавод |
| `CO2-ice` | Сухой лёд | CO₂ | C + O₂ | T < −50°C | 5 C + 9 O₂ | Химзавод |
| `N2-ice` | Замёрзший азот | N₂ | N₂ | T < −150°C | 10 N₂ | — (используется напрямую) |
| `CH4-ice` | Метановый лёд | CH₄ | C + H₂ | T < −150°C | 2 C + 6 H₂ | Химзавод |
| `NH3-ice` | Аммиачный лёд | NH₃ | N + H₂ | T < −100°C | 8 N + 12 H₂ | Химзавод |

### 4.2 Бонус температуры для ледодобывающей станции

| Температура поверхности | Бонус | Доступные соединения |
|------------------------|-------|---------------------|
| > +50°C | 0.0 — строить нельзя | — |
| 0°C — +50°C | 0.3 | H₂O-ice (полярные шапки) |
| −50°C — 0°C | 0.7 | H₂O-ice, CO₂-ice |
| −150°C — −50°C | 1.0 | H₂O-ice, CO₂-ice, NH₃-ice |
| < −150°C | 1.2 | H₂O-ice, CO₂-ice, N₂-ice, CH₄-ice, NH₃-ice |

### 4.3 Цепочки переработки ледяных соединений

```
H₂O-лёд (10)  ──[Химзавод]──▶  H₂ (8) + O₂ (8)
CO₂-лёд (10)  ──[Химзавод]──▶  C (5) + O₂ (9)
N₂-лёд (10)   ──────────────▶  N₂ (10)          ← без переработки
CH₄-лёд (10)  ──[Химзавод]──▶  C (2) + H₂ (6)
NH₃-лёд (10)  ──[Химзавод]──▶  N (8) + H₂ (12)
```

## 5. Глубинные ресурсы

Бурильная установка (Drilling Rig) извлекает редкие элементы из глубоких слоёв планеты. Доступность зависит от уровня установки:

| Уровень бурильной установки | Доступные слои | Типичные ресурсы |
|-----------------------------|---------------|-----------------|
| 1–4 | Верхняя мантия | Y-ore (иттриевая руда), Zr-ore (циркониевая руда), Be-ore (бериллиевая руда) |
| 5–7 | Нижняя мантия | Ir-ore (иридиевая руда), Os-ore (осмиевая руда), Ru-ore (рутениевая руда), Rh-ore (родиевая руда), Pd-ore (палладиевая руда), Hf-ore (гафниевая руда), Ta-ore (танталовая руда), Nb-ore (ниобиевая руда), In-ore (индиевая руда), Nd-ore (неодимовая руда), Ce-ore (цериевая руда), La-ore (лантановая руда), Dy-ore (диспрозиевая руда) |
| 8–10 | Ядро | Re-ore (рениевая руда), геотермальные источники |

> **Почему Y и Ba — глубинные, а не шахтные?** Иттрий и барий — редкоземельные элементы, встречающиеся преимущественно в глубоких магматических породах и пегматитах. Их концентрация в поверхностных рудах ничтожна. Для рецепта сверхпроводника (YBCO) нужны именно глубокие залежи, что делает сверхпроводники позднетейковым ресурсом.

> **Новое в версии 2.0:** добавлены редкоземельные руды нижней мантии — `In-ore` (индий, ур. 5), `Nd-ore` (неодим, ур. 5), `Ce-ore` (церий, ур. 5), `La-ore` (лантан, ур. 5), `Dy-ore` (диспрозий, ур. 6). Редкоземельные элементы критически важны для высокотехнологичного производства: Nd → постоянные магниты (электродвигатели, оружие), Dy → высокотемпературные магниты, In → сенсорные экраны и CIGS-панели, Ce → катализаторы, La → NiMH-батареи.

### 5.2 Связь с рецептом сверхпроводника

Рецепт сверхпроводника из ARCHITECTURE.md §3.2.2:

```
Сверхпроводник = Y(1) + Ba(2) + Cu(3) + O(4) + Энергия(15)
```

Цепочка добычи всех компонентов:

```
Y-ore (Бурильная уст. ур.2)  ──[Плавильня ур.5]──▶  Y
Ba-ore (Бурильная уст. ур.1) ──[Плавильня ур.4]──▶  Ba
Cu-ore (Шахта)               ──[Плавильня ур.1]──▶  Cu
H2O-ice (Ледодоб. ст.)       ──[Химзавод]────────▶  O
  или CO2 (Газ. экстрактор)  ──[Химзавод]────────▶  O
  или O2 (Газ. экстрактор)   ─────────────────────▶  O

Y + Ba + Cu + O ──[Химзавод ур.6]──▶  Сверхпроводник
```

> **Баланс:** сверхпроводник требует 4 разных источника (шахта, 2× бурильная, газовый/ледяной), что делает его производство сложной логистической задачей. Это обоснованно — сверхпроводники нужны для продвинутых модулей (ионный двигатель, щиты, варп-двигатель).

## 8. Структуры данных для программиста

### 8.1 Тип руды

```typescript
/** Определение руды / сырьевого ресурса */
interface OreDef {
  /** Уникальный ID (напр. 'Fe-ore', 'H2O-ice', 'CO2') */
  id: string;
  /** Человекочитаемое название */
  name: string;
  /** Тип сырья */
  type: 'metal_ore' | 'nonmetal_ore' | 'gas_compound' | 'ice_compound' | 'deep_ore';
  /** Здание добычи */
  sourceBuildingId: 'mine' | 'quarry' | 'gas_extractor' | 'drilling_rig' | 'ice_harvester';
  /** Главный элемент(ы), содержащиеся в руде */
  containedElements: {
    elementId: string;   // ID элемента (напр. 'Fe')
    yield: number;       // выход из 10 ед. руды (напр. 8)
  }[];
  /** Минимальный уровень здания добычи */
  minSourceLevel: number;
  /** Здание переработки */
  processingBuildingId: 'smelter' | 'chemical_plant' | 'refinery' | null;
  /** Минимальный уровень здания переработки (null если переработка не нужна) */
  minProcessingLevel: number | null;
  /** Энергозатраты на переработку за единицу */
  processingEnergyCost: number;
  /** Время переработки (тиков) */
  processingTime: number;
}
```

### 8.3 Маппинг для генерации залежей

При генерации планеты залежи (deposits) должны создавать **руды**, а не чистые элементы. Исправление к багу Р3-01:

```typescript
// БЫЛО (неправильно):
deposit.elementId = element.id;  // 'Fe' — чистый элемент

// СТАЛО (правильно):
deposit.oreId = getOreForElement(element.id);  // 'Fe-ore' — руда

function getOreForElement(elementId: string): string | null {
  const ORE_MAP: Record<string, string> = {
    Fe: 'Fe-ore', Si: 'Si-ore', Ti: 'Ti-ore', Al: 'Al-ore',
    Cu: 'Cu-ore', Ni: 'Ni-ore', Cr: 'Cr-ore', W: 'W-ore',
    Co: 'Co-ore', Au: 'Au-ore', Pt: 'Pt-ore', U: 'U-ore',
    Li: 'Li-ore', V: 'V-ore', C: 'C-ore', S: 'S-ore',
    Mn: 'Mn-ore', Zn: 'Zn-ore', Sn: 'Sn-ore', Pb: 'Pb-ore',
    Mo: 'Mo-ore', Ag: 'Ag-ore', Cd: 'Cd-ore', Se: 'Se-ore',
    K: 'K-ore', B: 'B-ore', F: 'F-ore',
    Y: 'Y-ore', Ba: 'Ba-ore',
    Zr: 'Zr-ore', Be: 'Be-ore',
    In: 'In-ore', Nd: 'Nd-ore', Ce: 'Ce-ore', La: 'La-ore', Dy: 'Dy-ore',
    Ir: 'Ir-ore', Os: 'Os-ore', Ru: 'Ru-ore', Rh: 'Rh-ore',
    Pd: 'Pd-ore', Hf: 'Hf-ore', Ta: 'Ta-ore', Nb: 'Nb-ore', Re: 'Re-ore',
  };
  return ORE_MAP[elementId] ?? null;
}
```

> **Для атмосферных и ледяных элементов** (H, He, N, O): залежи не создаются. Вместо этого газовый экстрактор и ледодобывающая станция производят соединения на основе состава атмосферы/льда планеты (см. разделы 3 и 4).
