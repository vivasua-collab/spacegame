## Критические расхождения (блокируют разработку)

### Р1-01: Газовый гигант имеет 127 гексов вместо 0 — застройка возможна

- **Документация (03-planets.md §1.1, §2.1, §3.2):** «Застройка поверхности — Невозможна (0 гексов)», «Газовый гигант | 25 000+ | 0 (застройка невозможна)»
- **Код (planet-types.ts):** `gas_giant: size='huge', hexCount=127, terrainWeights={plains:40}`
- **Влияние:** Код генерирует 127 гексов равнин для газового гиганта и размещает на них ресурсные залежи. Газовый экстрактор (`size: ['small','medium','large','huge']`) может быть построен на газовом гиганте. Противоречит фундаментальному правилу: газовые гиганты не имеют твёрдой поверхности. UI отобразит гекс-сетку для газового гиганта, вводя игрока в заблуждение.

### Р1-02: Два документа описывают разные системы размеров планет

- **Документация (03-planets.md §2.1, §3.2):** Карликовая=6 гексов, Малая=12, Средняя=24, Крупная=36, Газ.гигант=0
- **Документация (04-buildings.md §1.1):** Крошечная=19 гексов, Малая=37, Средняя=61, Большая=91, Огромная=127
- **Код (planet-types.ts):** tiny=19, small=37, medium=61, large=91, huge=127 — совпадает с 04-buildings.md
- **Влияние:** Два документа противоречат друг другу. Код следует 04-buildings.md, но 03-planets.md описывает совершенно другую (меньшую) систему. Любой разработчик, читающий 03-planets.md, получит неверные данные о количестве гексов.

### Р1-03: Отсутствуют 2 типа звёзд (красный гигант, пульсар) из документации

- **Документация (02-stars.md §2.1):** 12 типов звёзд: O, B, A, F, G, K, M + STAR_WD, STAR_RG, STAR_NS, STAR_PULSAR, STAR_BH. Красный гигант и пульсар имеют полные спецификации (температура, масса, светимость, влияние на планеты).
- **Код (star-types.ts, types.ts):** 10 типов: O, B, A, F, G, K, M + white_dwarf, neutron, black_hole. Нет `red_giant` и `pulsar`.
- **Влияние:** Генератор не может создать систему с красным гигантом или пульсаром, хотя они описаны как особые игровые объекты с уникальными механиками (сброс оболочки, пульсирующее излучение).

### Р1-04: Несоответствие ID звёзд между документацией и кодом

- **Документация (02-stars.md):** `STAR_O`, `STAR_B`, `STAR_A`, `STAR_F`, `STAR_G`, `STAR_K`, `STAR_M`, `STAR_WD`, `STAR_RG`, `STAR_NS`, `STAR_PULSAR`, `STAR_BH`
- **Код (types.ts StarType):** `'O' | 'B' | 'A' | 'F' | 'G' | 'K' | 'M' | 'white_dwarf' | 'neutron' | 'black_hole'`
- **Влияние:** Ни один ID из документации не совпадает с кодом для специальных типов. Если разработчик попытается использовать `STAR_WD` из документации, он не найдёт его в коде. Это также блокирует сериализацию/десериализацию — формат сохранений будет несовместим.

### Р1-06: AI-фракции полностью отсутствуют в коде

### Р1-07: Двойные/тройные звёздные системы не реализованы

- **Документация (ARCHITECTURE.md §3.2.1):** Химия, Физика, Инженерия, Биология, Военные науки, Ксеноархеология
