# SpaceGame

> Однопользовательская космическая 4X-стратегия / песочница в реальном времени

## О проекте

SpaceGame — однопользовательская космическая стратегия, вдохновлённая **«Полдень XXI века»** и **Aurora 4X**. Исследуйте процедурно генерируемую галактику, колонизируйте планеты, проектируйте корабли, ведите дипломатию с AI-фракциями и достигайте победы — научной, экономической, военной или дипломатической.

### Ключевые особенности

- **Процедурная галактика** — 100–2000 звёздных систем, уникальных при каждом seed
- **Глубокая колонизация** — гексагональная сетка застройки, бонусы местности, атмосфера и жизнь
- **Сложная экономика** — многоуровневый крафт: руда → металл → компонент → модуль → корабль
- **Модульные корабли** — конструируйте из десятков модулей: корпус, двигатель, оружие, щиты
- **Дерево технологий** — 72 технологии в 6 ветках, с уровнями и зависимостями
- **AI-фракции** — 5 уникальных цивилизаций с экономикой, военной стратегией и дипломатией
- **Управление временем** — пауза, x1/x5/x15/x50, отдача приказов во время паузы

### Назначение папок

| Папка | Назначение | Статус |
|-------|-----------|--------|
| `docs/` | Финальная проектная документация. Содержит полные спецификации всех систем игры. | Завершена (требуется синхронизация с кодом — см. `doc_temp/audit_final.md`) |
| `doc_temp/` | Промежуточная документация: ранние черновики, результаты аудитов, временные заметки. Не удалять — содержит историю анализа. | Активно используется |
| `checkpoints/` | Система чекпоинтов: отслеживание этапов разработки по правилам из `RULES.md`. | Активно используется |
| `src/core/` | Ядро движка: типы, PRNG, EventBus, GameLoop. Чистый TS, без зависимостей от React. | MVP завершён |
| `src/data/` | Справочные данные игры: элементы, звёзды, планеты, здания, рецепты. Формат TS (не JSON). | MVP завершён (8 зданий, 20 элементов) |
| `src/galaxy/` | Генератор галактики: спиральная генерация, гекс-сетка, Jump Points. | MVP завершён |
| `src/economy/` | Экономический движок: добыча, производство, энергия, крафт. | MVP завершён (требует фиксов по аудиту) |
| `src/stores/` | Zustand-сторы для управления состоянием. | MVP завершён |
| `src/components/game/` | Игровые UI-компоненты: карты, экраны, диалоги. | MVP завершён |
| `src/components/ui/` | shadcn/ui библиотека (не изменять). | Сторонняя |
